Multiple Decisions

We can also make multiple decisions at once. For example, let's change the drawing color based on the size of num:

Let's step through the program:

  1. We create a turtle for drawing.
  2. For each item in the list,
    1. If num is less than 50, change the color to blue. If it's between 50 and 100, change the color to red. Otherwise, make the color black.
    2. Move forward the distance num from our list,
    3. Turn our turtle to the right 93 degrees, and

The variable, num, takes each value in the list of the numbers. We first check that if num is less than 50, we change the color to blue. Only if num is not less than 50, then we go onto to the elif statement, which checks if num is less than 100. If it is, then the color is changed to red. If both tests fail, then we go to the else which change the color to black.
