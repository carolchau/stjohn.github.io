Here's some useful turtle commands:

import turtle includes the library filled with the turtle drawing commands.
tess = turtle.Turtle() creates a turtle named tess.
for i in range(7):
    (add code here) 		repeats indented code 7 times.
tess.forward(10) moves tess forward 10 steps.
tess.left(90) turns tess left 90 degrees.
tess.color("blue") turns tess blue (need the quotes around the color name, so that Python interprets it as the word, "blue", instead of as a variable (such as "tess").
tess.shape("turtle")    changes tess to have a turtle shape.
