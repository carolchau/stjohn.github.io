Problem Set 3

CMP 230: Programming Methods I
Lehman College, City University of New York
Spring 2010

This problem set is based on the material in Chapter 3: Decisions & Loops.

You should complete 130 points worth of programs (with an additional 13 points extra credit possible).

  1. Too Hot (5 points)

    Create a world with a snowman and stove in it (found in the People and Kitchen galleries, respectively). In the my first method, add in an If/Else statement that the snowman jumps if he is within 2 meters of the stove. He should do nothing otherwise. Test your decision statement by placing your snowman in different spots in the world.

  2. Shrinking Bunny (10 points)

    Create a world with a bunny in it. In the my first method, add in an If/Else statement that 30% of the time the bunny doubles in size and the rest of the time the bunny shrinks to half of its original size. Play the program several times to show the affect.

  3. Morning Dancing (10 points)

    Create a world with a dancer in it (several are in the People gallery). Write an If/Else statement that causes the dancer to spin in place two revolutions if it's currently morning.

    (Hint: Use the Is AM world function (see Appendix B of the book for more details)).

  4. Changing Colors (5 points)

    Create a world with a snowman and stove in it (found in the People and Kitchen galleries, respectively). In the my first method, add in an If/Else statement that the snowman turns red is within 2 meters of the stove, otherwise he turns blue. You can control the color of the snowman by using color from its list of properties.

  5. Bike at Intersection (5 points)

    Modify the Intersection2 program (available on the book's CD or class Blackboard site) so that the bike turns left 75% of the time and right 25% of the time.

  6. Random Roo (10 points)

    Create a world with a kangaroo under a tree. Have the kangaroo move a random distance forward. Next use an If/Else statement that causes the kangaroo to spin in place if he is farther than a meter from the tree.

  7. Bruce the Shark (5 points)

    Modify the Shark2 program (available on the book's CD or class Blackboard site) so that Bruce potentially rises to the surface every other lap.

  8. Hopping Forever (5 points)

    Using your kangaroo from Lab 3, create a world where your kangaroo hops forever.
    Hint: Use a While true statement.

  9. Collision (5 points)

    Modify the Collision program (available on the book's CD or class Blackboard site) so that when the trucks collide, a crashing sound is played and the cement truck's right front wheel breaks loose and continues to roll down the street.

  10. Stop Hop (10 points)

    Using your kangaroo world from above, add in a tree and have your kangaroo hop towards tree as long as the distance is greater than 3 meters.
    Hint: Use the distance behind proximity function (see Appendix B).

  11. Falling Object (10 points)

    Exercise 3.5 (p 63): Write a While statement that causes an object of your choice (some possible ones: a ball, a skydiver, a water balloon, a meteor, etc.) to move down one meter as long as it is above a specific object on the ground (e.g. a target, a person, a house, etc.).
    Hint: Use the is above proximity function (see Appendix B).

  12. Ten Twirls (5 points)

    Exercise 3.9: Write a loop statement that causes an object (e.g. a skater, a bunny, a car, etc) to spin in place 10 times.

  13. Count Down (15 points)

    Exercise 3.8: Write a While statement updates the value of a variable called seconds and displays it in a 3D text object called countdown. Start seconds at 10 and loop until it reaches 0.
    Hint: use the "what as text" function to convert your variable into text that can be displayed on the screen.

  14. Odometer (20 points)

    Modify the SpeedingCar program (available on the book's CD or class Blackboard site) so that it displays the distance the car has traveled while it is moving. Continuously display the result on a 3D text object that stays visible in the upper portion of the display window.
    Hint: use the "what as text" function to convert your variable into text that can be displayed on the screen.

  15. Biplane (10 points)

    PP 3.2 (p 64): Create an Alice world in which a Biplane from the Vehicles gallery flies in from the right side of the viewing area, makes seven consecutive veritcal loops as it moves across the sky, then flies off to the left. Its propeller should be turning the entire time.

  16. Frog Race (15 points)

    PP 3.5: Create an Alice world in which two frogs race toward a finish line. Use a loop in which both frogs hop forward in each iteration. Determine the distance of each hop randomly. End the animation when one of the frogs crosses the finish line. The Frog class is in the Animals gallery.

  17. Bouncing Ball (20 points)

    PP 3.6: Create an Alice world in which a ball rolls off a table, bounces on the ground, and eventually comes to rest. Use the ToyBall class from the Sports gallery to create the ball. Use a loop to control the bounces, decreasing the height of the rebound by half every time. Move the ball away from the table slightly on each bounce. Continue the bounces until the rebound height decreases to an insignificant amount.

Submitting Your Problem Set

You should submit 130 points worth of problems for grading by the deadline (see class schedule for date). To submit a program, you need to demonstrate (during office hours or lab) your program and explain how it works.

If you complete extra problems by the deadline, up to 13 additional points can be counted towards extra credit.
