Problem Set 5

CMP 230: Programming Methods I
Lehman College, City University of New York
Spring 2010

This problem set is based on the material in Chapter 5: Lists and Arrays.

You should complete 100 points worth of programs (with an additional 10 points extra credit possible).

  1. School Of Fish (5 points)

    Modify the SchoolOfFish world so that after swimming a bit, a shark is added to the front of the school list and begins to swim with the school.

  2. Ballet (5 points)

    PP 5.1 (p 96): Create an Alice world in which a group of ballerinas dance in unison, making appropriate poses and turns.

  3. Rockettes (5 points)

    PP 5.2: Create an Alice world in which a line of Rockettes performs various kicks and turns in unison. Also include some moves that are executed in sequence down the line.

  4. Dominoes (5 points)

    Create a world with at least ten objects arranged like dominoes (for example, you could use the Box class from the Shapes gallery. These items should all be in a list. When the user clicks anything in the world, the first item should fall over and strike the second item, which should fall over and strike the third item, and so on.

  5. Even Rockettes (10 points)

    Create an Alice world in which contains a line of Rockettes. Have every other rockette kick, executed in sequence down the line.

  6. Shortest-to-Tallest (15 points)

    Modify the Alice world, SortingDemo.a2w (available on Blackboard), to have an array of ten different people (it currently has 8). In the world, the program should sort the people, shortest to tallest (instead of the tallest to shortest in the program).

  7. Soldiers (10 points)

    Create an Alice world in which a group of 12 soldiers march in three rows of four soldiers each. After marching forward a certain distance, have the soldiers halt and salute. Use the ToySoldier class from the People gallery.

  8. Airport (10 points)

    Create a world that simulates an airport with a control tower and a runway. A group of airplanes should be lined up, ready for take-off. Store the airplane objects in a list. One at a time, the airplanes should take off and fly away into the distance.

  9. Select-A-Penguin (20 points)

    Create an Alice world with 10 penguins in it and store the penguin objects in an array. When the user clicks on the 0 key, the first penguin in the list should jump up and down. When the user clicks on the right arrow key, the next penguin in the array should jump up and down. Similarly, if the user clicks on the left arrow key, the previous penguin in the array should jump up and down.

    For example, if the current penguin is the third one in the list, and the user clicks on the right arrow, the fourth penguin in the list jumps up and down. If the current penguin is the last one in the list, and the user clicks on the right arrow, the first penguin in the list jumps up and down. Similarly, If the current penguin is the first one in the list, and the user clicks on the left arrow, the last penguin in the list jumps up and down.

    Hint: Create a number variable called current that holds the index of the current penguin. Increment or decrement that variable depending on which key the user clicks.

  10. Bank Teller (20 points)

    PP 5.4: Create an Alice world that simulates people waiting in line at a bank teller window. Randomly determine the amount of time it takes to serve each customer. When a customer leaves, each person in line should move forward one space. Display the wait time for each customer when the customer is finally served.

  11. Tank (20 points)

    PP 5.9: Create an Alice world that plays a game in which a tank shoots at planes that are crossing in the distance. Use the Tank class in the Vehicles gallery. Allow the user to rotate the turret right and left and change the angle of the barrel up or down using the arrow keys. Use the space bar to "fire" a ball. Store various types of planes in an array to use as the targets, moving them across the sky at various heights and distances. Keep track of the number of hits.

Submitting Your Problem Set

You should submit 100 points worth of problems for grading by the deadline (see class schedule for date). To submit a program, you need to demonstrate (during office hours or lab) your program and explain how it works.

If you complete extra problems by the deadline, up to 10 additional points can be counted towards extra credit.
