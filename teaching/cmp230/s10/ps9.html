Problem Set 9

CMP 230: Programming Methods I
Lehman College, City University of New York
Spring 2010

This problem set is based on the material in Chapter 9 (Inheritance and Polymorphism).

You should complete 50 points worth of programs (with an additional 5 points extra credit possible).

  1. ShapeMaker (20 points)

    Add the Oval and the Circle classes to the ShapeMaker program (available on Blackboard) and modify the controls that allow them to be drawn.

  2. Monetary Coin (20 points)

    PP 9.2 (p 203): Create a class called MonetaryCoin that is derived from the Coin class presented in Chapter 6 (available on Blackboard). Store a value in the monetary coin that represents its value and add a method that returns its value. Create a driver class whose main method instantiates several MonetaryCoin objects and computes their total value. Demonstrate that the monetary coin inherits its parent's ability to be flipped.

  3. Happy Face (20 points)

    Make a new class, called HappyFace that extends the Oval (or Circle) class from Chapter 9 (available on Blackboard). Your class should have methods that allow the user to draw a happy face on the screen. To demonstrate your class, add it to the ShapeMaker program and modify the controls so that it also can be drawn.

  4. Cross (25 points)

    Derive a new class, Cross, from the BoundedShape class from Chapter 9 (available in Blackboard). Your class should draw a cross (i.e. an "X") inside the bounding box. Demonstrate your new class with a program that draws several crosses.

Submitting Your Problem Set

You should submit 50 points worth of problems for grading by the deadline (see class schedule for date). To submit a program, you need to demonstrate (during office hours or lab) your program and explain how it works.

If you complete extra problems by the deadline, up to 5 additional points can be counted towards extra credit.
