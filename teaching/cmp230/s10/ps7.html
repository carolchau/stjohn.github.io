Problem Set 7

CMP 230: Programming Methods I
Lehman College, City University of New York
Spring 2010

This problem set is based on the material in Chapters 6 (Transition to Java) and 7 (Events in Java).

You should complete 150 points worth of programs (with an additional 15 points extra credit possible).

  1. Lots of Circles (10 points)

    Make a copy of the Snowman program (available from the CD or Blackboard) to use as the base for this program. The program should allow users to enter the number of circles they would like drawn on the screen in a text box and the circles should be drawn in panel. You will need a label, Number of Circles, a text box, and an associated listener.

    Hint: Look at how the "stars" are drawn in the original program, and modify the drawing to make visible circles.

  2. Person Class (10 points)

    Finish the Person class (file available on Blackboard) that will be used to list contact information for people in an address book. Comments above each method describe what the method should do. Also available on Blackboard is a driver, PersonDriver to demonstrate the Person class.

  3. Push Counter (15 points)

    Create a new project for the PushCounter program from the book (available from the CD or Blackboard). The program consists of the files: PushCounter.java, PushCounterPanel.java, and ButtonListener.java. Make sure it runs in Eclipse.

    Next, modify the program in the following way:

    &bull Add a button and a corresponding listener that will zero the counter (that is, when your new button is pushed, the counter will be set to zero and that will be displayed in the text field).
    &bull Add a button and listener that decrements the counter and updates the text field accordingly.
  4. Shapes (15 points)

    Make a program with three different buttons, labelled, Square, Circle, and Oval. Your program should also have listeners for each of these shapes. When the user clicks on one of the buttons, the corresponding shape should be drawn on the panel.

  5. Toggle (15 points)

    Make a copy of the PushCounter program to use for this program. Your program should have a single button on a black background. If the button is pushed and the background is black, the background turns yellow. Otherwise, if the button is pushed and the background is yellow, turn the background black.

  6. Image Flicker (10 points)

    Extend the ImageFlicker program (consisting of ImageFlicker.java and ImageFlickerPanel.java) to show five images of your choice. The user should be able to view the images by typing 1 through 5 to see the individual images, or the space bar to see a random image.

  7. Drawing Cards (15 points)

    Design and implement an application that displays a button and two labels. Every time the button is pushed, the first label displays a random number between 1 and 13, and the second label displays randomly one of the following: Clubs, Diamonds, Hearts, or Spades

    Hint: First, write the code to do first label (with random number between 1 and 13). Then, generate a random number between 1 and 4. If the number is 1, change the label to be Clubs. If the number is 2, change the label to be Spades, etc.

  8. Random Walk (15 points)

    Write a program to simulate the walk of a drunkard in a square street grid. Place a simulated person at the center of the window, denoted by a filled circle. For 100 times, have the simulated person randomly pick a direction (east, west, north, south), move one block in that direction, and redraw the circle.

    So that you can watch the progression of the walk, have the person wait 0.1 seconds between each step. To wait in Java, you can use the Thread.sleep() method. For example, the following will make the current thread sleep for 1 second:

    Thread.sleep(1000);

    Include a button labeled Play Again that clears the window and simulates another random walk.

  9. Mouse Odometer (20 points)

    PP 7.1: Design and implement an application that serves as a mouse odometer, continually displaying how far, in pixels, the mouse has moved (while in the program window). Display the current odometer value using a label.
    Hint: Compare the current position of the mouse to the last position and use the distance formula to determine how far the mouse has traveled. Given two points (x,y) and (u,v), the distance formula is Math.sqrt( (x-u)*(x-u) + (y-v)*(y-v) ).

  10. Space Ship (10 points)

    Extend the SpaceShip program (consisting of Spaceship.java, SpaceshipPanel.java, and SpaceshipFlight.java) to allow the user to change the size of the space ship. If the user types L, the space ship should be large size; M should be medium size; and S should be small size.

  11. Traffic Light (15 points)

    PP 7.5 (p 151): Design an application that draws a traffic light and uses a key press to change the state of the light. Each time the space key is pressed, change the state of the light.

  12. Capture the Creature (30 points)

    PP 7.10: Design and implement an application that plays a game called Catch-the-Creature. Use an image (or a simple object) to represent the creature. Have the creature appear at a random location for a random duration, then disappear and reappear again. The goal is to "catch" the creature by pressing the mouse button while the mouse pointer is on the creature image.

    Create a separate class to represent the creature, and include in it a methd that determines whether the location of the mouse click corresponds to the current location of the creature. Display acount of the number of times the creature is caught.
