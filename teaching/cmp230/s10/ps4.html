Problem Set 4

CMP 230: Programming Methods I
Lehman College, City University of New York
Spring 2010

This problem set is based on the material in Chapter 4: Events.

You should complete 100 points worth of programs (with an additional 10 points extra credit possible).

  1. Event Tutorial (5 points)

    Open the Alice animation system and choose Open World from the File Menu (this should automatically open when you open Alice). Click on the Tutorial tab, and choose Tutorial 3 (penguin chorus). Work through the entire tutorial.

  2. Jumping Frogs (10 points)

    PP 4.2 (p 83): Create an Alice world showing two frogs prepared to jump. When the user clicks on one of the frogs using the mouse, that frog should jump a random distance forward.

  3. Kitchen (10 points)

    PP 4.3 (p 83): Create an Alice world showing a kitchen environment with various appliances (refrigerator, oven, microwave, toaster) and other appropriate items created from classes in the Kitchen gallery. Allow the user to click on the appliances to open and close their doors and otherwise interact with the kitchen objects.

  4. Dragon (10 points)

    Create an Alice world with a dragon in it. When you type `F' the dragon should breathe fire. When you type `S', a puff of smoke should appear. Both the fire and smoke should disappear after half a second.

  5. Kangaroo (10 points)

    Using the kangaroo (with the hop method with parameters from Lab 3), create a new world where the user controls how far the kangaroo jumps. When the user clicks on a number between 1 and 9, the kangaroo jumps forward that many meters. For example, if the user clicks on the '6' key, the kangaroo jumps forward 6 meters.

  6. Horse (10 points)

    Create an Alice world in which a horse is relaxing in a field. Use the Horse class in the Animals gallery. Define four different basic actions for the horse (sway its tail, dip its head, whinny, etc.) When the user clicks any key, have the horse perform one of the four actions, at random.

  7. Lighthouse (15 points)

    PP 4.4: Create an Alice world in which a lighthouse stands on the edge of some water. Add a light source to the lighthouse that constantly turns. Allow the user to click on the lighthouse with the mouse to turn off the light and to turn it back on again. Use the LightHouse class from the Beach gallery and the DirectionalLight from the Lights gallery.

  8. Helicopter Simulator (15 points)

    PP 4.5 (p 84): Create an Alice world in which a Helicopter from the Vehicles hovers , blades constantly turning. Allow the user to:

    move the helicopter up and down using the up and down arrow keys,
    turn the helicopter right or left using the right and left arrow keys, and
    move forward using the space bar

    Build an interesting world to fly around to make it obvious the helicopter is moving.

  9. Motorboat Race (20 points)

    PP 4.7: Create an Alice world in which a motorboat is controlled using three arrow keys: the up arrow moves the boat forward and the right and left arrows turns the boat. Set up a series of obstacles and gates to demonstrate the maneuverability of your boat.

    Add a timer. It should track the time it takes for the boat to reach a finish line and display it as a stop watch on the screen.

  10. Ring Toss (20 points)

    PP 4.9: Create a ring catching game using a torus and cone created from the Shapes gallery. The ring should fall from the sky while the user moves the cone along the ground with the mouse. The user's objective is to catch the ring with the cone. Following each attempt, reset the ring's position. Stop when the user has caught 10 rings.

Submitting Your Problem Set

You should submit 100 points worth of problems for grading by the deadline (see class schedule for date). To submit a program, you need to demonstrate (during office hours or lab) your program and explain how it works.

If you complete extra problems by the deadline, up to 10 additional points can be counted towards extra credit.

Logging Out

Before leaving the lab, you need to log out and shut down your computer. Remember to pick up any files you sent to the printer and to take any personal belongings. It is difficult to get back into the lab later (since it's locked when not in use), so, it's worth making sure you haven't forgotten anything.
