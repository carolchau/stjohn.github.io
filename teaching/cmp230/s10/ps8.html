Problem Set 8

CMP 230: Programming Methods I
Lehman College, City University of New York
Spring 2010

This problem set is based on the material in Chapter 8 (Arrays and Lists in Java).

You should complete 100 points worth of programs (with an additional 10 points extra credit possible).

  1. Rocket Drawing (10 points)

    Modify the Rocket program (available from the book or Blackboard) so that there is another rocket, also drawn using arrays and polylines.

  2. Colored Dots (10 points)

    Modify the Dots program (available from the book or Blackboard) so that the first dot created is drawn in yellow, the newest dot is drawn in red, and all other dots are drawn in green.

  3. Disappearing Dots (10 points)

    Modify the Dots program so that after 10 dots have been created, the oldest one disappears whenever a new one is created.

  4. Number Counts (15 points)

    PP 8.4 (p 174): Create a program that reads an arbitrary number of integers that are in the range 0 to 9, inclusive, and counts how many occurrences of each are entered. After all input has been processed, print all of the values that were entered one or more times with the number of occurrences of each.

  5. Numbers (25 point)

    Create a program that asks the user to read in 10 integers and prints the maximum, the total and the average of the list of numbers to the screen.

    For example, if the user entered: 

    10 12 16 13 4 5 21 2 6 1
    the maximum entered is 21, then total is 90 and the average (total divided by number of items in list) is 9.

  6. Polyline (20 points)

    PP 8.9 (p 175): Create a program that allows the user to make a polyline shape dynamically using mouse clicks. Each mouse click adds a new line segment from the previous point to the location of the mouse click. Include a button below the drawing area to clear the current polyline and begin another.

    Hint: start with the Dots program and have it draw lines between a point and the previous one in the list.

  7. Person List (20 point)

    Using the Person.java class from the last problem set, create an array of 10 people. Use a loop to ask for the first and last name of each person and then print out the list of people (using the toString() method).

  8. Select-A-Circle (25 points)

    (The idea of this program is similar to Problem 5.9 Select-A-Penguin.)

    Using the ImageFlicker program as a guide, create a new program that will show 10 blue circles in a row (like the 10 penguins in a row in the Select-A-Penguin program). When the user clicks on the 0 key, the first circle in the list should turn red. When the user clicks on the R key, the next circle in the array should turns red (and the previous one returns to being blue). Similarly, if the user clicks on the L key, the previous circle in the array should turn red (and the rest should be blue).

    For example, if the current circle is the third one in the list, and the user clicks on the right arrow, the fourth circle in the list turns red. If the current circle is the last one in the list, and the user clicks on the right arrow, the first circle in the list jumps up and down. Similarly, If the current circle is the first one in the list, and the user clicks on the left arrow, the last circle in the list turns red.

    Hint: Create a number variable called current that holds the index of the current circle. Increment or decrement that variable depending on which key the user clicks. The paintComponent() method then paints all circles blue, followed by painting the current circle red.
