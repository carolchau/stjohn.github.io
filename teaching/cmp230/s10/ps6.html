Problem Set 6

CMP 230: Programming Methods I
Lehman College, City University of New York
Spring 2010

This problem set is based on the material in Chapter 6: Transistion to Java.

We will be using the Eclipse IDE to program in Java. A good reference on using Eclipse by Daniel Liang. Remember before starting a project to go to File -> Switch Workspace and change the workspace to your USB drive, so, that your work is saved on your drive instead of the machine's hard drive.

You should complete 200 points worth of programs (with an additional 20 points extra credit possible).

  1. Tutorial (5 points)

    To start today, open up Eclipse and work through the first tutorial, "Create a Hello World Application". It is on the terse side, but you should end up with a program that says "Hello World" in the console.

  2. Circles (10 points)

    If you haven't already done so, switch workspaces to be your USB drive. Create a new project, Circles, which will be used for several projects. Next, import (File -> Import) the Circles files from the book (available off the CD or off Blackboard). Run the program (you should see a new window with 3 colored circles) to make sure it was imported correctly. Modify your program to have 4 different unfilled circles.

  3. Face (15 points)

    In the Circles project, copy the DrawCircles.java and CirclePanel.java to new files called DrawFace.java and FacePanel.java.

    Use these new files to create a program that will display a snowman. Some useful things to keep in mind:

    &bull The names of the file must match the names of the class contained in them.
    &bull Other than changing the name (of file and class), you will not have to change DrawFace.java.
    &bull Your face should be made up of at least 3 circles for eyes, mouth, and outline.
  4. Bullseye (10 points)

    Modify the Bullseye program from the book (available from the CD or on Blackboard) so that:

    &bull the largest ring is white and the others alternatve accordingly, and
    &bull there are approximately twice as many rings that are each half as thin.
  5. Coins (10 points)

    Modify the CoinFlip program (available from the book or Blackboard) so that it allows up to 10,000 flips.

  6. Random Circle (10 points)

    In your Circles project, copy the DrawCircles.java and CirclePanel.java to new files called DrawRandom.java and RandomPanel.java. Use these new files to create a program that will draw a single circle at a random location each time.

    Hint: Use the random number generator to generate the x and y coordinates for your circle (an example of using random numbers can be found in the Coin programs and the LineUp (StickFigure) programs.

  7. Tails (10 points)

    Modify the CoinFlip program (available from the book or Blackboard) so that the Coin class uses an isTails method instead of isHeads.

  8. Verify (15 points)

    Ex 6.1 (p 120): Write a Java program containing a while loop that verifies that the user has entered a positive integer value.

  9. Odd Numbers (15 points)

    Ex 6.3 (p 120): Write a Java program that uses a for loop to print out the odd numberes from 1 to 99, inclusive.

  10. Stick Figures (10 points)

    Using the StickFigure program from the book (available on Blackboard), do the following:

    &bull there are two rows of people, and
    &bull the tallest person is half of what it was before.
  11. Highest Number (15 points)

    Ex 6.4 (p 120): Write a Java program that reads in 10 integer values from the user and then prints the highest value entered.

  12. 100 Circles (15 points)

    PP 6.1 (p 121): Using the LineUp and Circles classes from the book (available from the CD or on Blackboard), create a Java program that draws 100 circles of random size, color, and location. Ensure that in each case the entire circle appears in the visible area of the panel.

  13. 3 Heads (15 points)

    PP 6.2 (p 121): Using the Coin class from this chapter, create a Java program that flips two coins until one of them comes up heads three times in a row.

  14. Parallel Lines (10 points)

    PP 6.3 (p 121): Write a Java program that displays 20 horizontal, evenly spaced lines of random length.

  15. Star (15 points)

    PP 6.6: Create a class called Star that represents a five-pointed star of a particular size and color. Then write a program that creates and draws four stars of various sizes at different locations.

  16. Account (25 points)

    PP 6.7 (p 121): Create a class called Account that represents a basic bank account, keeping track of the account balance and providing methods to process deposits and withdrawls. Print an error message if an attempt is made to withdraw more than the current balance. Include a toString method that summarizes the state of the account. Create a driver program that creates two Account objects and processes various transactions.

  17. Dice (20 points)

    PP 6.8 (p 121): Create a class called Die that represents a single six-sided die (as in the the singular of dice). Then write a program that creates and rolls two Die objects 500 times, and counts how many times snake eyes (both dice showing 1) comes up.

Submitting Your Problem Set

You should submit 200 points worth of problems for grading by the deadline (see class schedule for date). To submit a program, you need to demonstrate (during office hours or lab) your program and explain how it works.

If you complete extra problems by the deadline, up to 20 additional points can be counted towards extra credit.

Logging Out

Before leaving the lab, you need to log out and shut down your computer. Remember to pick up any files you sent to the printer and to take any personal belongings. It is difficult to get back into the lab later (since it's locked when not in use), so, it's worth making sure you haven't forgotten anything.
