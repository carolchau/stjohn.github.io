Problem Set 10

CMP 230: Programming Methods I
Lehman College, City University of New York
Spring 2010

This problem set is based on the material in Chapter 10 (Exceptions and I/O).

You should complete 50 points worth of programs (with an additional 5 points extra credit possible).

  1. Divider (15 points)

    Write a program that asks the user to enter two numbers, and then divides the first by the second and outputs the answer.

    Catch any exceptions that might be thrown (i.e. when the second number is 0) and handle it gracefully.

  2. Letter Count (15 points)

    Modify the letter count program (available on Blackboard) to ask the user for a name of a file. Your program should then count the letters in the file and outputs the number of times each letter occurs.

  3. Doublespacing (25 points)

    Write a program that asks the user for the name of an input file and the name of an output file. The program should create a double spaced version of the input file in the output file.

    For example, if the input file, letter.txt is: 

    This is my letter to the world,
That never wrote to me,
The simple news that Nature told,
With tender majesty.

Her message is committed
To hands I cannot see;
For love of her, sweet countrymen,
Judge tenderly of me!

    And the user specifies the output file double.txt should be:

    This is my letter to the world,

That never wrote to me,

The simple news that Nature told,

With tender majesty.



Her message is committed

To hands I cannot see;

For love of her, sweet countrymen,

Judge tenderly of me!

If you complete extra problems by the deadline, up to 5 additional points can be counted towards extra credit.
