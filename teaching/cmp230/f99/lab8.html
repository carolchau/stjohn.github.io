Account Class

Last week, we used a class Employee that we wrote in lecture. This week, we're going to write a similar class, but for bank accounts.

Start by creating a file called, Account.java which will hold the Account class. This class should contain the holder's name, address, social security number, and current balance. Your class should also have methods to print, get the current balance, deposit and debit money to the account:

  • public void print()
  • public double getBalance()
  • public void deposit(double d)
  • public void debit(double d)

After you have your class file, you need to write a program that uses your Account class. Take the testEmployee file and modify it to also use Accounts. Set up accounts for two people, deposit money to their accounts, print the accounts. Also, debit money from their account and print the accounts.
