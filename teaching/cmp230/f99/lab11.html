Today's lab is on inheritance (see Chapter 6 of the textbook). We will be using projects in JBuilder, since projects are a good way to keep multiple files in a program together. See lab 10 for more details about projects.

Savitch's Figure class

Savitch has written a parent class Figure class, from which he has extended to make a class Box, for drawing boxes, and Triangle for drawing triangles. Today, we'll use his classes to draw a simple picture of a house.

First, set up a new project called Lab11. In this project, you should add the files:

  • Figure.java (the parent class)
  • Box.java (a child class)
  • Triangle.java (a child class)
  • GraphicsDemo.java (the class that contains the main())
These files are in the book (p 306-319), available in the StudentData folder in the labs, and also on the web.

Compile and run the program. What does it draw? Modify GraphicsDemo.java so that the program draws a simple house, using only boxes and triangles. Note: using these simple figures, it is not possible to "nest" one box inside another.
