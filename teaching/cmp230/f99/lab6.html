Mystery Output

What does the following code fragment do? 
int i, j;
for ( i = 0; i < 5 ; i++ )
{
    for ( j = 0; j < 5 ; j++)
    {
        if ( (i+j)%2 == 0 )
        {
            System.out.print("*");
        }
        else
        {
            System.out.print("=");
        }
        System.out.println();
    }
}
To figure this out, put the above code in a program, compile, and run it. What does the %2 do? How would the output differ if the test condition in the if statement was changed to: 
if ( j%2 == 0 )
Try it and find out.

Stars and Stripes

Modify your program from above, to print out: 
*****
=====
*****
=====
*****
What did you change, and why?
