PlayingCard Class

In lecture, we wrote a program that takes 5 PlayingCards and sorts them by their rank. In today's lab, we will use the same class for our objects, but this program will sort the cards, first by suit, then by rank.

For example, if your hand was: 

    Jack Hearts  2 Spades  4 Clubs  6 Hearts  3 Diamonds
Your program would print: 
    4 Clubs  3 Diamonds  6 Hearts  Jack Hearts  2 Spades

Start by copying PlayingCard.java and SortCards.java from the webpage. Load these files into J-Builder, compile and run them.

Modify the SortCards.java program to sort according to the above criteria. Do not modify the PlayingCard.java program.

  • First, make your program sort by suit. Remember, to compare strings, you use equals instead of ==.
  • Then add in an extra test in case the suit is the same for two of the cards.
