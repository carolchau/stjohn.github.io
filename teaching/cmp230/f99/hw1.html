P1. Programming Exercise 2, p 27.

P2. Write a program that asks the user to enter a phrase. Your program should then print out the phrase three times and leave the phrases on the screen until the user hits a key.
For example: 
    Please enter a phrase:
Hello world!
You said:
Hello world!
Hello world!
Hello world!
Please enter any key to end the program.
Modify the program on p 18. Note instead of reading in a single character, you need to read in an entire line.
Hint: To print out a string variable named mystring, you could use the following statement: 
    System.out.println(mystring);
