Today's lab is on using projects in JBuilder. To demonstrate their use, we'll continue working with the playing card example.

Projects

Projects, in JBuilder, are a way of grouping files for a program together. For example, our Employee programs had two files: one was the "blueprint" for making objects of type Employee and the other file contained the main program. To keep these files together, we can create a project which will make a separate folder to hold these programs.

Today, we'll create a project to hold the files for our card game. Open JBuilder, and click on the New Project option under the file menu. A dialog box will appear in the middle of the screen. The first line is where JBuilder will store your files. You should change it to say: 

a:\myprojects\bridgegame\bridgegame.jpr
The next lines in the dialog box are for your name, a title for the program, and a short description. Fill these in, and click on Finish.

It will take JBuilder a few seconds to set up the project. When it has, you will see a list of files on the left (which should look similar to before) and a title page on the left for your project.

Next, you want to add files to your project. Click on the plus (+) sign above the list of files. This will open a new file for you. Call the first file PlayingCard.java and cut-and-paste the file from the webpage. (PlayingCard.java) Repeat with DeckOfCards.java (DeckOfCards.java) and Bridge.java (Bridge.java).

Next, add to the end of Bridge.java two lines of code that tell the user to "Press any key to continue" and saves their answer in Junk. This will keep the runtime window from disappearing before you can view the output.

To compile and run your program using Projects is the same as before. If you have troubles, see Lab 1 or ask for help.

Bridge Class

Compile and run your Bridge program. It deals out the deck of cards to 4 players (North, South, East, and West) and prints out their hands. For today, you need to sort each player's hands and print them out.

Use the code from the SortCards program to sort each hand. Note: you will need to modify it slightly since it sorts an array called hand of 5 cards, and you need to sort 4 different arrays (one for each player) of 13 cards each.

Do not modify the PlayingCard.java or the DeckOfCards.java files.
