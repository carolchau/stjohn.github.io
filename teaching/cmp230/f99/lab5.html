Maximum Score

The program ExamAverager.java from the textbook (p 133) asks the user to enter exam scores and prints out the average.

Copy this program into J-Builder, compile and run it. What happens if you enter no scores? How do exit the program? What do you have to type? How many exams can you average?

Modify the program, to print out, not only the average, but the maximum score for the exam. You will need another variable to hold the maximum score. In that variable, store the highest score seen so far (you can test if another score is higher by using an if-statement).

Minimum Score

After you add the code that prints out the high score, add code that will also print out the lowest score for the exam. This is done in similar way to finding and printing the maximum score.
