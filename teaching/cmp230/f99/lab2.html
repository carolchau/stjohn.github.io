Dollars and Cents

First, write a program that takes as input an integer, which will be the amount of cents the user has. Your program should output, the amount entered, in dollars and cents format.
For example, if the user entered 1234, your program should output: 
    You have $12.34

Hours, Minutes, and Seconds

Second, write a program that takes as input an integer, which will be the amount of seconds left in class. Your program should output the number of hours, minutes and seconds left in class.
For example, if the user entered 1234, your program should output: 
    There are 0 hours, 20 minutes, and 34 seconds left in class.
