Employee Class

First, we're going to look at the class Employee we wrote in lecture. It's at Employee.java and a program that uses it: testEmployee.java.

Copy each of these programs and load them into JBuilder. (There's several ways to do this. An easy way is to cut-and-paste into new windows in J-Builder. You can also save each as a file (on the floppy drive) in Netscape, and then open the files from J-Builder).

Build first the Employee class, and then the testEmployee class. Run the testEmployee class.

Adding Another Object

Modify your program from above to include another object of type Employee. Call this new variable, theUser. Add in Java code that asks the user for their name, address, phone number, and salary. Store this information in the object theUser and then use the member function print to show the user what they entered.
