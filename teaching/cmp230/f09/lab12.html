General Information About the Lab

For information about getting started, see Lab 1.

Today, we continue with Java and Eclipse. The focus is on using the control structures for repetition and loops we learned about in Alice in Java.

Bullseye

Modify the Bullseye program from the book (available from the CD or on Blackboard) so that:

  • the largest ring is white and the others alternatve accordingly, and
  • there are approximately twice as many rings that are each half as thin.

Circles

PP 6.1 (p 121): Using the Circles class from the book (available from the CD or on Blackboard), create a Java program that draws 100 circles of random size, color, and location. Ensue that in each case the entire circle appears in the visible area of the panel.

Star

PP 6.6: Create a class called Star that represents a five-pointed star of a particular size and color. Then write a program that creates and draws four stars of various sizes at different locations.

Stick Figures

Using the StickFigure program from the book (available on Blackboard), do the following:

  • there are two rows of people, and
  • the tallest person is half of what it was before.
