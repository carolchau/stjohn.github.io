Today's lab is an introduction to loops and repetition.

Repetition

For today's lab, we will do a series of short programs involving while and loop statements:

Hopping Forever

Using your kangaroo from Lab 3, create a world where your kangaroo hops forever.
Hint: Use a While true statement.

Stop Hop

Using your kangaroo world from above, add in a tree and have your kangaroo hop towards tree as long as the distance is greater than 3 meters.
Hint: Use the distance behind proximity function (see Appendix B).

Falling Object

Ex 3.5 (p 63): Write a While statement that causes an object of your choice (some possible ones: a ball, a skydiver, a water balloon, a meteor, etc.) to move down one meter as long as it is above a specific object on the ground (e.g. a target, a person, a house, etc.).
Hint: Use the is above proximity function (see Appendix B).

Ten Twirls

Ex 3.9: Write a loop statement that causes an object (e.g. a skater, a bunny, a car, etc) to spin in place 10 times.

Count Down

Ex 3.8: Write a While statement updates the value of a variable called seconds and displays it in a 3D text object called countdown. Start seconds at 10 and loop until it reaches 0.
