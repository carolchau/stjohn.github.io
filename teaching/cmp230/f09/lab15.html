Image Flicker

Extend the ImageFlicker program (consisting of ImageFlicker.java and ImageFlickerPanel.java) to show five images of your choice. The user should be able to view the images by typing 1 through 5 to see the individual images, or the space bar to see a random image.

Space Ship

Extend the SpaceShip program (consisting of Spaceship.java, SpaceshipPanel.java, and SpaceshipFlight.java) to allow the user to change the size of the space ship. If the user types L, the space ship should be large size; M should be medium size; and S should be small size.

Traffic Light

PP 7.5 (p 151): Design an application that draws a traffic light and uses a key press to change the state of the light. Each time the space key is pressed, change the state of the light.

Extra Credit

Capture the Creature

PP 7.10: Design and implement an application that plays a game called Catch-the-Creature. Use an image (or a simple object) to represent the creature. Have the creature appear at a random location for a random duration, then disappear and reappear again. The goal is to "catch" the creature by pressing the mouse button while the mouse pointer is on the creature image.

Create a separate class to represent the creature, and include in it a methd that determines whether the location of the mouse click corresponds to the current location of the creature. Display acount of the number of times the creature is caught.
