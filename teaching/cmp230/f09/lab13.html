Today, we continue with Java. This is the first lab focusing on events.

Push Counter

Create a new project for the PushCounter program from the book (available from the CD or Blackboard). The program consists of the files: PushCounter.java, PushCounterPanel.java, and ButtonListener.java. Make sure it runs in Eclipse.

Next, modify the program in the following way:

  • Add a button and a corresponding listener that will zero the counter (that is, when your new button is pushed, the counter will be set to zero and that will be displayed in the text field).
  • Add a button and listener that decrements the counter and updates the text field accordingly.

Shapes

Make a program with three different buttons, labelled, Square, Circle, and Oval. Your program should also have listeners for each of these shapes. When the user clicks on one of the buttons, the corresponding shape should be drawn on the panel.

Toggle

Make a copy of the PushCounter program to use for this program. Your program should have a single button on a black background. If the button is pushed and the background is black, the background turns yellow. Otherwise, if the button is pushed and the background is yellow, turn the background black.
