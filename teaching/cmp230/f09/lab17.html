Number Counts

PP 8.4 (p 174): Create a program that reads an arbitrary number of integers that are in the range 0 to 50, inclusive, and counts how many occurrences of each are entered. After all input has been processed, print all of the values that were entered one or more times with the number of occurrences of each.
