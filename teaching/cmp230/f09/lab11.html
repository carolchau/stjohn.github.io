General Information About the Lab

For information about getting started, see Lab 1.

Today's lab is our first using Java and the integrated development environment (IDE) Eclipse. The majority of the lab will focus on getting comfortable using Eclipse.

Eclipse is an open source program and is available on the CD at the back of the book, as well as at http://www.eclipse.org/downloads.

Tutorials

A good reference on using Eclipse by Daniel Liang.

Remember before starting a project to go to File -> Switch Workspace and change the workspace to your USB drive, so, that your work is saved on your drive instead of the machine's hard drive.

To start today, open up Eclipse and work through the first tutorial, "Create a Hello World Application". It is on the terse side, but you should end up with a program that says "Hello World" in the console.

Circles

If you haven't already done so, switch workspaces to be your USB drive. Create a new project, Circles, which will be used for several projects over the next couple of labs.

Next, import (File -> Import) the Circles files from the book (available off the CD or off Blackboard). Run the program (you should see a new window with 3 colored circles) to make sure it was imported correctly.

Modify your program to have:

  • 4 different circles
  • one circle unfilled
  • all circles have different size and color than the original.

Snowman

In the Circles project, copy the DrawCircles.java and CirclePanel.java to new files called DrawSnowmen.java and SnowmanPanel.java.

Use these new files to create a program that will display a snowman. Some useful things to keep in mind:

  • The names of the file must match the names of the class contained in them.
  • Other than changing the name (of file and class), you will not have to change DrawSnowmen.java.
  • Your snowman should be made up of at least 3 circles for the body and include a face or hat.
