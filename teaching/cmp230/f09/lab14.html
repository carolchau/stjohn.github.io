Snowman

Make a copy of the Snowman program (consisting of Snowman.java, SnowmanPanel.java, and SnowmanDrawPanel.java) to use as the base for this program. Your program should have all the features of the Snowman program, and also have an additional set of radio buttons that allows you to change the color of the snowman's hat. You should give the user at least three hat colors to choose from, and you should create an event listener for this new set of radio buttons.

Filling Choice

Make a copy of the Circles program (available from the CD or Blackboard) to use as the base for this program. Add a check box, labeled Filled?. Your program should draw three circles, either filled, if the check box is checked, or unfilled otherwise. Your program should also have a listener for the check box (see the Snowman program for examples of check boxes).

Circles

Make a copy of the Snowman program (available from the CD or Blackboard) to use as the base for this program. The program should allow users to enter the number of circles they would like drawn on the screen in a text box and the circles should be drawn in panel. You will need a label, Number of Circles, a text box, and an associated listener. Instead of writing a new Circle class, reuse the one from Circles program.

Red Light, Green Light

PP 7.3 (p 151): Design and implement an program whose background changes color depending on where the mouse is located. If the mouse pointer is on the left half of the program window, display red; if it is on the right half, display green.

Extra Credit

Mouse Odometer

PP 7.1: Design and implement an application that serves as a mouse odometer, continually displaying how far, in pixels, the mouse has moved (while in the program window). Display the current odometer value using a label.
Hint: Compare the current position of the mouse to the last position and use the distance formula to determine how far the mouse has traveled. Given two points (x,y) and (u,v), the distance formula is Math.sqrt( (x-u)*(x-u) + (y-v)*(y-v) ).
