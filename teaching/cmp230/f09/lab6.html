General Information About the Lab

For information about getting started, see Lab 1.

Today's lab focuses on more complicated worlds with loops and repetition.

Biplane

PP 3.2 (p 64): Create an Alice world in which a Biplane from the Vehicles gallery flies in from the right side of the viewing area, makes seven consecutive veritcal loops as it moves across the sky, then flies off to the left. Its propeller should be turning the entire time.

Frog Race

PP 3.5: Create an Alice world in which two frogs race toward a finish line. Use a loop in which both frogs hop forward in each iteration. Determine the distance of each hop randomly. End the animation when one of the frogs crosses the finish line. The Frog class is in the Animals gallery.

Bouncing Ball

PP 3.6: Create an Alice world in which a ball rolls off a table, bounces on the ground, and eventually comes to rest. Use the ToyBall class from the Sports gallery to create the ball. Use a loop to control the bounces, decreasing the height of the rebound by half every time. Move the ball away from the table slightly on each bounce. Continue the bounces until the rebound height decreases to an insignificant amount.
