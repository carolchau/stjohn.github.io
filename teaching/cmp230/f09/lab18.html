Drawing Playing Cards

Add to the PlayingCard class (available on Blackboard), a method that will draw a playing card on the screen: 

  public void draw(x,y)

where x and y are the coordinates of the upper left corner of the playing card.

To demonstrate your new method, create a driver program that has a panel with several cards displayed on it.

Sorting by Suit

Change the SortCards.java program (available on Blackboard) so that it first sorts first by suit, then by rank. For example, the current sort program, if you entered:

  2 spades 3 diamonds 5 clubs 7 spades 8 clubs
would output: 
  2 spades 3 diamonds 5 clubs 7 spades 8 clubs

Your modified program should group cards by their suit (clubs, diamonds, hearts, spades) first, then by their rank. So, your program would output for the above input:

  5 clubs 8 clubs 3 diamonds 2 spades 7 spades

Sorting Hands

Fill in the missing parts of Bridge.java (available on Blackboard). That is, after the hands have been dealt and printed, sort the hands and print them again.
(Hint: the file SortCards.java has code that might be useful.)
