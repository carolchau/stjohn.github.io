General Information About the Lab

For information about getting started, see Lab 1.

Today's lab is the first on using lists (called ArrayLists in Java).

Colored Dots

Modify the Dots program so that the first dot created is drawn in yellow, the newest dot is drawn in red, and all other dots are drawn in green.

Disappearing Dots

Modify the Dots program so that after 10 dots have been created, the oldest one disappears whenever a new one is created.

Polyline

PP 8.9 (p 175): Create a program that allows the user to make a polyline shape dynamically using mouse clicks. Each mouse click adds a new line segment from the previous point to the location of the mouse click. Include a button below the drawing area to clear the current polyline and begin another.

Hint: start with the Dots program and have it draw lines between a point and the previous one in the list.
