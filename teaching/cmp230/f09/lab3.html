Using Parameters

Modify your world with a kangaroo from last lab so that the hop method accepts a parameter that specifies the jumping distance. Demonstrate your new method by having the kangaroo hop several times.

Programming Project 2.3 (p 38): Create an Alice world that shows a search and rescue opertation in which a helicopter flies in a pattern above the ocean, occasionaly stopping to hover for a while as if looking for something. Eventually the helicopter swoops in to save a person who was adrift. Create a hover method the helicopter that accepts the number of seconds it should hover as a parameter. Create a search method and a rescue method to handle those aspects of the operation.
