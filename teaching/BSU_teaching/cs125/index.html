Using the Computers:

  • Departmental Computer Policy
  • Hints on using unix
  • The accounts for this class live on emerald. You cannot log on to any of the other gemstones (i.e. agate, ruby, turquoise, etc.). If the graphics terminal you wish to use has a message welcoming you to another gemstone, turn the terminal off and on. This will cause the terminal to reboot and will bring up the list of gemstones (from which you can choose emerald).
  • You can also use emerald from the Multipurpose Classroom Facility General Computer Lab. Ask one of the assistants for the handout on Reflection X. The handout gives step-by-step instructions. Insert emerald.idbsu.edu for the name of the host.
  • If you're interested in working from home, there are several programs you will need:
    • telnet: This is a program that allows you to log on to a server from your home computer. You can type commands and edit files (using vi) but cannot do graphics. Telnet is all you need for the first couple of weeks of the course.

      • If you are using
      • linux: This is included with the kernel.
      • MacOS: A good version is BetterTelnet, available from http://www.cstone.net/~rbraun/mac/telnet/.
      • Windows: The built-in version is buggy (in can be accessed by typing telnet: into your browser). You can find free and shareware at http://www.tucows.com/.
    • ftp: If you would like to edit files at home (using Word, for example), you need a file transfer program to move the files to emerald. (More on obtaining ftp later).
    • jdk: This is the java compiler we're using for the class. It is available free for most operating systems from http://www.java.sun.com. You will only need this if you plan to compile programs at home and then transfer them back to emerald. For this, you will also need the library the book uses which can be downloaded from http://www.horstmann.com/ccj.html.
