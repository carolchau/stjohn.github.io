  4. * Write a program that adds line numbers to a file and stores the result in an output file. For example, if your input file is: 
    hello
helloo
hellooo
    Your output file would be: 
    1   hello
2   helloo
3   hellooo
    Your program should ask the user for the name of the input and output files. It should also use the formatting functions to make the line number take at most 4 spaces and always be flush to the left column.
  5. * Write a program counts the number of characters (not including carriage returns), the number of words, and the number of lines in a file. Ask the user to enter the name of the file and print your results to the screen.

    You can check your work by using the UNIX command wc on your input file. For example, if you type 

    wc template.html
    at the "bash$" prompt, the response is: 
    33     116     810 template.html
    where 33 is the number of lines, 116 is the number of words, and 810 is the number of characters in template.html. Note that it is difficult to read this output without prior knowledge of what the columns should be. Your program should label the output for the user.
