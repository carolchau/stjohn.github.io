Reversing a Linked List
In this lab, you need to write a void-function that takes a linked list of integers and reverses the order of its nodes. The function will have one call-by-reference parameter that is a pointer to the head of the list. After the function is called a pointer will point to the head of a linked list that has the same nodes as the original list, but in the reverse order they had in the original list. Note that your function will neither create nor destroy any nodes. It will simply rearrange the nodes.

You should use the following definitions for the nodes and pointers to nodes: 

struct Node
{
    int data;
    Node *link;
};

typedef Node* NodePtr;
Your linked list should follow the style that we discussed in class and that is illustrated in the book: e.g. there is a head node that contains no data and the ending node's link is set to NULL.

Your program should prompt the user to enter as many integers as the user wishes, it should then store the numbers in a linked list, print out the linked list, call the reversing function, and print out the linked list again (this time in reversed order).
