Basic HTML

Part of today's lab is to create your own web page. Most web pages are written using the language, HTML, which stands for Hyper Text Markup Language. There are many similarities between C++ and HTML. For example, in C++, curly braces "{" and "}" come in pairs and mark the beginning and end of functions or blocks of code. For HTML, the beginning and end of the title (which appears at the top of the window) is indicated by "<title>" and "</title>", and the beginning and end of a string written in a boldface font by "< b >" and "< /b >". The title of this file is written as: 
	<title> Lab 4 for Computer Engineering 6, SCU </title>

Every web page should begin with it's title, followed by the body statement. Like with the title, there's a beginning of the body ("<body>") and an end of the body ("<body>"). In between these two, you should put the body, or main part, of your webpage. The body statement for this page has some extra "adjectives" which also specify the color of the page and the links: 

	<body bgcolor="ffffff" link="003399" vlink="236b8e">
These extra descriptions occur only in the beginning body statement, not the ending one. "bgcolor" specifies the background color of the page ("ffffff" is the hexadecimal representation for white), "link" gives the color of links, and "vlinks" gives the colors links turn after you have looked at them at least once. The end of body statement is just: 
	</body>
Comments in HTML are indicated by surrounding your comment with "<!--" and "-->".

Colors in HTML

Colors are represented by a 6-digit hexadecimal number. A hexadecimal number ranges from 0 to 15 (10 through 15 are represented by A through F), much like a binary digit ranges from 0 to 1, or a decimal digit ranges from 0 to 9..

"ffffff" represents the color white, while "000000" represents the color black. Try changing the background color of your page to get blue, red, and green. You can do this by trial-and-error, looking at the source for other pages that use those colors, or by using a webpage editor such as Composer.

Computing Change Program

Write a program that tells what coins to give out for any amount of change from 1 cent to 99 cents. For example, if the amount is 86 cents, the output would be something like the following: 
        86 cents can be given as
        3 quarter(s) 1 dime(s) and 1 penny(pennies)
Use coin denominations of 25 cents (quarters), 10 cents (dimes), and 1 cent (pennies). Do not use nickel and half-dollar coins. Your program will use the follwing function (among others): 
void compute_coin(int coin_value, int& number, int& amount_left);
// Assumes: 0 < coin_value < 100; 0 <= amount_left < 100.
// The function sets number equal to the maximum number of coins
// of coin_value cents that can be obtained from amount_left cents.
// amount_left is decreased by the value of the coins
// (i.e. number*coin_value).
Include a loop that lets the user repeat this computation for new input values until the user says he or she wants to end the program. Hint: use integer division and the % operator to implement the above function.
