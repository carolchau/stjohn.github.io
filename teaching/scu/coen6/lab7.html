Simplifying Fractions

Any function that computes a new fraction (including constructors) should store the fraction in its simplest form. For example, "3/6" should be stored as "1/2". To do this, you need to find the greatest common divisor of the numerator and denominator and divide both by this number. Use the following "helper" function to do this: 
int gcd(int u, int v)
{
    while (u > 0)
    {
        if ( u < v )
            swap(u,v);
        u = u-v;
    }
    return v;
}
Note that you need also to include in your program the swap function we discussed in lecture.
