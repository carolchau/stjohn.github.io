Here's a simple applet that will display a parametrized function. It assumes that the ranges of the x and y are [-1,1]. To display a different function, download the file orbits.java and change the values for x and y.



To compile a java program in the lab, type: 
~stjohn/java/bin/javac orbits.java
To display your program, you need to include it in a webpage, using the following lines in your html file: 
<applet code="orbits.class" height=400 width=400>
</applet>
This assumes that the orbits.class file is in the same directory as your html file.
