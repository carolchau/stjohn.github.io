  • Homework 6 is now available. It relies on several chapters from Abiteboul, Hull, and Vianu, which are available from Halaulani, in the department office.
  • Some of the more bizarre system problems this semester:
    • Passwords that are longer than 8 characters are allowed on onyx but cause problems when propagated to the linux boxes.
      Solution: Use shorter passwords.
    • The informix scripts count on undefined shell variables being intepreted as null.
      Solution: Remove the line: 
      set -u
      from your .profile. This line is included in the default .profile that comes with your account.
    • Running the database gives a read error when trying to connect.
      Solution: This problem is related to the setup of Informix on our system. We're using NFS to share onyx's disk between the linux boxes (and thus avoiding maintaining 5 separate copies of Informix).
