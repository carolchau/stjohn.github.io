Problem Set 9

CMP 326: Programming Methods II
Lehman College, City University of New York
Fall 2010

This problem set covers material from Chapter 11 (Recursion).

You should complete 50 points worth of programs (with an additional 5 points extra credit possible).

  1. Blinking (10 points)

    Write a graphical program that every second changes the screen from red to green. The program should be very simple, and there is no need for any user interface (ie buttons or text fields).

  2. Pop Up Window (15 points)

    Write a graphical program that has a main window and an additional window that pops up a window for 5 seconds and disappears. Use any of the graphical programs from earlier in the semester as the base and see the counting primes program as a way to time the length that something occurs.

  3. Time Out (15 points)

    Write a program that times out if the user does not enter anything in 20 seconds.
    (Hint: use the counting primes program as a base).

  4. Rolling Ball (15 points)

    Write a graphical program where a ball "rolls" across the screen.
    (Hint: every tenth of a second refresh the screen with a circle in a new position to give the appearance of moving across the screen.)

  5. Poker (40 points)

    Implement the poker program from the Wellesley College page: http://cs.wellesley.edu/~ecom/problem_sets/ps1.html.

Submitting Your Problem Set

You should submit 50 points worth of problems for grading by the deadline (see class schedule for date). To submit a program, you need to demonstrate (during office hours or lab) your program and explain how it works.

If you complete extra problems by the deadline, up to 5 additional points can be counted towards extra credit.

Logging Out

Before leaving the lab, you need to log out and shut down your computer. Remember to pick up any files you sent to the printer and to take any personal belongings. It is difficult to get back into the lab later (since it's locked when not in use), so, it's worth making sure you haven't forgotten anything.
