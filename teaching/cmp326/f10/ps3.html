This problem set covers material (one dimensional arrays and sorting) from Chapter 7 (arrays).

You should complete 70 points worth of programs (with an additional 7 points extra credit possible).

  1. Temperature (10 points)

    Exercise 7.1 (p 524): Write a program in a class NumberAboveAverage that counts the number of days that the temperature is above average. Read ten temperatures from the keyboard and place them in an array. Compute the average temperature and then count and display the number of days on which the temperature was above average.

  2. Character Frequency (10 points)

    Exercise 7.5 (p 524): Write a program in a class CharacterFrequency that counts the number of times a digit appears in a telephone number. Your program should create an array of size 10 that will hold the count for each digit from 0 to 9. Read a telephone number from the keyboard as a string. Examine each character in the phone number and increment the appropriate count in the array. Display the contents of the array.

  3. Ledger (15 points)

    Exercises 7.6-7 (p 525): Create a class Ledger that will record the sales for a store. It will have the attributes:

    • sale: an array of double values that are the amounts of all sales
    • salesMade: the number of sales so far
    • maxSales: the maximum number of sales that can be recorded
    and the following methods:
    • Ledger(max): a constructor that sets the maximum number of sales to max
    • addSale(d): adds a sale whose value is d
    • getNumberOfSales(): returns the number of sales made
    • getTotalSales(): returns the total value of sales
    • getAverageSale(): returns the average value of all sales
    • getCountAbove(v): returns the number of sales that exceeded v in value

    Write a driver program that demonstrates these methods.

  4. Strictly Increasing (10 points)

    Exercise 7.8 (p 525): Write a static method isStrictlyIncreasing(double[] in) that returns true if each value in the given array is greater than the value before it, or false otherwise. Write a driver program that demonstrates this method.

  5. Sorting Characters (10 points)

    Exercise 7.15 (p 526): Write a static method for selection sort that will sort an array of characters. Write a driver program that demonstrates this method.

  6. Sorting Subarrays (10 points)

    Exercise 7.16 (p 526): Overload the method selectionSort that appears in Listing 7.10 so that it sorts an array whose indices range from first to last, where 0 <= first <= last and last is less than the length of the array.

  7. BubbleSort (15 points)

    Problem 7.4 (p 530): Add a method bubbleSort to the class ArraySorter (available from textbook or Blackboard) that performs a bubble sort of an array. The buuble sort algorithm examines all adjacent pairs of elements in the array from the beginning to the end and interchanges an two elements that are out of order. Each interchange makes the array more sorted than it was, until it is entirely sorted. The algorithm in pseudocode follows: 

    Bubble sort algorithm to sort an array a

Repeat the following until the array a is sorted:
for (index = 0; index < a.length - 1; index++)
  if (a[index] > a[index+1])
    Interchange the values of a[index] and a[index+1]
    The bubble sort usually requires more time than other sorting methods
