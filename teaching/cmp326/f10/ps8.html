  5. Double Factorial (15 points)

    The double factorial of a number, written x!! is very similar to the factorial: 

    0!! = 1
1!! = 1
2!! = 2
3!! = 3 * 1
4!! = 4 * 2 
5!! = 5 * 3 * 1
6!! = 6 * 4 * 2 * 1
...
If n is odd, n!! = n * (n-2) * ... * 1  
If n is even, n!! = n * (n-2) * ... * 0
    Write a method that implements double factorial and place in a test program that allows the user to enter a number and then prints out the double factorial of the number.

  7. Quicksort (20 points)

    Implement the following version of the recursive sort, QuickSort (from http://en.wikipedia.org/wiki/Quicksort). Here is the pseudocode of the two methods: 

      function partition(array, left, right, pivotIndex)
     pivotValue := array[pivotIndex]
     swap array[pivotIndex] and array[right] // Move pivot to end
     storeIndex := left
     for i  from  left to right - 1 // left ≤ i < right 
         if array[i] ≤ pivotValue
             swap array[i] and array[storeIndex]
             storeIndex := storeIndex + 1
     swap array[storeIndex] and array[right] // Move pivot to its final place
     return storeIndex


  procedure quicksort(array, left, right)
     if right > left
         select a pivot index //(e.g. pivotIndex := left+(right-left)/2)
         pivotNewIndex := partition(array, left, right, pivotIndex)
         quicksort(array, left, pivotNewIndex - 1)
         quicksort(array, pivotNewIndex + 1, right)

    Write a driver program that demonstrates that your sort.
