This problem set covers material (multidimensional arrays and arrays in graphics) from Chapter 7 (arrays).

You should complete 150 points worth of programs (with an additional 15 points extra credit possible).

The first problems in this set rely on a code base developed by Richard Wicentowski and Tia Newhall from the Department of Computer at Swarthmore College. To use their code, do the following:

  • Download the code from: http://www.cs.swarthmore.edu/~newhall/imagemanip/java_staringpt.zip.
  • Import the java files into a new project in Eclipse.
  • Copy a jpeg file into the project directory in your workspace (not the src directory but its parent directory).
  • Before running the program, configure the "Run" to use your jpeg file. To do this, open up the "Run Configurations" option from the "Run" menu. Choose the tab labeled "Arguments" and put the name of your file here.
  • Read the documentation from Swarthmore:
    Overview of their software and useful documentation, and
    Specifics about programs below, including demos.
