  2. This problem was harder than we thought, and has been dropped from the assignment. You do not need to turn this one in. Write a program that will display an image of your choice. When the user clicks on the image, you should display the blue channel of the image only. When the user clicks on the image a second time, the image displays in black and white. Note that a simple way to display an image in black and white is to display the average in each channel. That is, for every pixel with RGB values, (r, g, b), displaying ( (r+g+b)/3, (r+g+b)/3, (r+g+b)/3 ) will give a black and white version of the image.

  3. Write a program that show a ball moving across the screen. The user should click on the screen for the starting point, and the ball should move from left to right across the screen for 1000 iterations of a loop.
    Notes:
