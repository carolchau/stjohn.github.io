This assignment focuses on computing with numbers from Chapter 3.

Submit the following programs via Blackboard:

  1. Write a program that takes the length of a class in minutes and returns the number of hours and minutes. For example, if the user entered, 160, your program would output, 2 hours 40 minutes. Note that you should use "hours" and "minutes" for all inputs (even in the cases where it is grammatically incorrect).
  2. Chapter 3, #2 (pizza). Note that the input, the price of pizza, is a decimal number.
  3. A function related to factorial is the double factorial, n!! = n(n-2)(n-4)...1 for odd numbers and n!! = n(n-2)(n-4)...2 for even numbers. For example, 7!! = 7*5*3*1 = 105. Modify the factorial.py (from the textbook or author's website) to calculate the double factorial.
  4. Chapter 3, #14 (average)
  5. Chapter 3, #15 (approximating pi)
