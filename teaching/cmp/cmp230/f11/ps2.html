This assignment focuses on the material in the first two chapters.

Submit the following programs via Blackboard:

  1. Chapter 1, #4: Modify the chaos program (available from the author's website) so that it prints out 20 values instead of 10.
  2. Chapter 2, #2: Modify the avg2.py program (Section 2.5.3 from the book) to find the average of three exam scores.
  3. Chapter 2, #4: Modify the convert.py program (Section 2.2, and available from the author's website) so that it computes and prints a table of Celsius temperatures and the Fahrenheit equivalences every 10 degrees from 0C to 100C.
  4. Chapter 2, #6: Suppose you have an investment plan where you invest a certain fixed amount every year. Modify futval.py (available from the author's web page) to compute the total accumulation of your investment. The inputs to the program will be the amount to invest each year, the interest rate, and the number of years for the investment.
  5. Chapter 2, #9: Write a program that converts distances measured in kilometers to miles. One kilometer is approximately 0.62 miles.
