We will be using Blackboard throughout the semester to submit assignments, so, the primary goal of the first assignment is to become accustomed to using the system.

Using Blackboard

This course will use the on-line Blackboard system for submitting work electronically and on-line quizzes. Blackboard should be accessible through your CUNY portal account (http://portal.cuny.edu). If you have not used Blackboard in the past, information is available at the Computer Center. They can set up your account and have information sheets about using the system.

Timing Out: If the system times out and locks your attempt (happens rarely when the browser or PC crashes), send email to an instructor so they can clear the attempt so you can try again.

Work Submitted

After logging into the class webpage on Blackboard, go to Assignments and click on the link for the first assignment. A short set of questions will appear on the screen. Answer the questions, and a grade will appear on the screen. You may repeat the problem set as many times as you wish before the deadline.
