Problem Set 8
CMP 230: Programming Methods I
Lehman College, City University of New York
Spring 2012

Due Date: midnight, Friday, 4 May.

This assignment focuses on loops and booleans (Chapter 8).

Submit the following programs via Blackboard:

  1. Chapter 8, #3 (doubling investment)
  2. Chapter 8, #9 (fuel efficiency)
  3. Write a program that prompts the user for the names of an input file and an output file. Assume the input file consists of words separated by spaces. The program creates an output file containing the same words, but with all occurrences of your first name replaced by your initials.
  4. Write a function that takes as input a list of numbers and prints out the numbers on the list until it encounters a negative number. Write an accompanying program that demonstrates how your function works.
  5. Write a program that reads in a text file, infile.txt, and writes out the negative numbers to another file, outfile.txt. You may assume that every line of the input file consists of exactly one number.
For more information on using Blackboard, see the first problem set.
