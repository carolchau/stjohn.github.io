This assignment focuses on computing with numbers from Chapter 3.

Submit the following programs via Blackboard:

  1. Write a program that takes a length in inches and returns the number of feet and inches (recall that there are 12 inches to a foot). For example, if the user entered, 30, your program would output, 2 feet 4 inches. Note that you should use "feet" and "inches" for all inputs (even in the cases where it is grammatically incorrect).
  2. Chapter 3, #5 (coffee shop). Note that the input, the number of pounds of coffee ordered, is a decimal number.
  3. Chapter 3, #9 (area of a triangle)
  4. Chapter 3, #12 (cubes)
  5. Chapter 3, #13 (sums)
