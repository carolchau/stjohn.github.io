Installing Eclipse on a Mac

Eclipse is a multi-language software development environment. We will use it in CMP 230 with Python, and in CMP 326 with Java. It is a good tool to get acquainted with early.

Eclipse is obtained from http://eclipse.org/downloads/. You want Eclipse Classic, 64bit, unless you have an old computer. This downloads a tar file in your downloads folder which, when double clicked, expands into the eclipse folder. Drag this to your Applications folder.

When you start Eclipse, you will be asked for the name and location of a workspace folder. The two most likely places for location are in your Documents folder, and in the Eclipse folder itself. Either is fine, you choose. Eventually you will probably have two workspaces, one for Java if you take CMP326, and one for Python. Since Eclipse will start off ready for Java, you might call your first one javaworkspace. Another will be created for Python shortly, and you won't use the javaworkspace for quite a while.

Next, we set up Eclipse so it can work with Python as well as with Java. Before you do this, Python 3 must be installed. Instructions here. Then, configuring Eclipse to work with Python is described in http://www.rose-hulman.edu/class/csse/resources/Eclipse/eclipse-python-configuration.htm, and other places as well. Here is a short summary.

  1. From the File menu select Switch Workspace/other, and create a pythonworkspace.
  2. From the Help menu select Install New Software.
  3. Select Add.
  4. Enter http://pydev.org/updates under Location, and click OK.
  5. Back in the "Available Software" window, select the "PyDev for Eclipse" option. Do not select the "PyDev Mylyn Integration" flag.
  6. Click "Next", verify that PyDev is selected in the next window then select "Next" again.
  7. Accept the terms, and click Finish. This will probably require that Eclipse be restarted.
  8. From the Eclipse menu select Preferences, expand Pydev, and select Interpreter-Python.
  9. Click New. Type Python31 for Interpreter Name, and for Interpreter Executable, click Browse, and click your way through as follows. Begin with your hard drive, select Library, then Frameworks, then Python.framework, then Versions, then 3.1, then bin, then python3.1, and click Open.
  10. Click OK when you have the next menu. Use the default selections.
  11. Click OK, and that's it.
