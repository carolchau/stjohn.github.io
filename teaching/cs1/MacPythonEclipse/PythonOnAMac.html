Installing Python on a Mac

Macs come with Python installed, BUT, it is not the current version (it is version 2) and will not do for this class. You must install the most recent version, version 3.

Your Mac must have operating system Mac OS X 10.3 through 10.6. Then use your browser to go to http://www.python.org/download/. Click on Python 3.1.2 Mac OS X Installer Disk Image. This will create a disk image, visible on your desktop. Make sure you click on the right thing, and not on Python 2.7, which is the older version of Python. Open the disk image and click on Python.mpkg which will start the installer. Install on your hard drive, and then eject the disk image.

You should now have a folder called Python 3.1 in your Applications folder. Open it and click on Update Shell Profile.command.

At this point you should have Python 3 installed. You can now do any of the following, and you should try them all.

  1. You can run Python from a terminal window. In your Applications Folder, inside the Utilities Folder, is Terminal.app. You should add it to your dock. When you start the Terminal application you can type commands. Typing Python3 starts python going, and then you can interact with it. Notice the "3". If you just type Python you will get the older Python 2, which you don't want.
  2. From the terminal window you can type idle3 which will start the environment we will be using for the first part of the course. Once again, if you forget the "3" and just type idle, you will get something that uses the older version of python.
  3. In your Applications Folder there is a Python 3.1 Folder. In this there is IDLE.app. You should add this to your dock. Double clicking on it is another, and convenient, way to start the idle environment.
