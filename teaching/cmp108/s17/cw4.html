Getting Started with Folium

Folium is a Python package that uses the javascript Leaflet.js library to make beautiful interactive maps. Instead of popping up a matplotlib window, folium creates an .html file that you can open (and view interactively) with a browser. After each program runs, open the corresponding html file to see your map. Check to see if you have it already: 
	import folium
If not, go to the terminal, and download it: 
pip install folium

Work through the examples through adding markers (stop before the Vincent/Vega Markers section, since the sections after this require data files).

Challenges
