Using Variables and Constants

We build a bigger donut shop that allows the user to order other kinds of donuts. (You will need to download the files from Chapter 2 to complete this sections of the lab. Note that const does not work on Internet Explorer. If you are using IE, make sure you download the alternative version of this page.)

Add to the donut page:

  • another kind of donuts that can be ordered (at 50 cents a donut),
  • also allow the user to order bagels (at 75 cents a bagel),
  • following the format for cake and glazed donuts, include these two new options in the final total for the order.

Sandwich Shop, Revisited

Add to your 228 Sandwich Shop page some JavaScript that will test if the user entered their name in the form. If they didn't, bring up a dialog box that asks them to re-enter their name.

Hint: Reuse the JavaScript developed for the donut shop example above.
