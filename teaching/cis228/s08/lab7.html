Today's lab focuses on an extension of HTML called XHTML. As in last lab, we will focus on good form for our pages, verifying this by using an XHTML validator.

Using XHTML

Before starting the lounge and segway page exercises, you will need to download the files provided on the book's webpage to a floppy drive or USB keychain.  If you haven't already done so, the book's website is:
http://headfirstlabs.com/books/hfhtml/index.php
We're going to work with the index.html file in Chapter 7 (the segway example, revisited). 
  1. Add/change in the <DOCTYPE> tag at the top of the journal.html file to be strict XHTML .
  2. Use the validator, http://validator.w3.org, to check if the page meets standards.
  3. Remember that empty elements need to end in " />" instead of ">".
  4. Fix any mistakes (hint: inline elements must be in blocks) and try the validator again.

All-About-Me Website

The final activity for today's lab we will to choose a page that you've written for a previous lab and modify it to meet XHTML 1.0 standards.  Use the validator to verify that the page is in standard XHTML.
