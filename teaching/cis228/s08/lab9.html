Fonts

Next, we're going to look at some basic style related to fonts.  We'll work with the journal.html file in Chapter 9 (the segway example, revisited):
  1. In the journal folder for Chapter 9, create a file journal.css with:

    body {
       font-family: Verdana, Geneva, Arial, sans-serif;
    }
     
  2. Now, link your new file journal.css to journal.html and view the results in a browser.
  3. Make the body font small (using keywords) and the h1 heading 150% and h2 heading 120%.  View your results in a browser to make sure the font sizes have changed.
  4. To make the headings stand out less, change the font-weight to normal.
  5. Also, add italic font to quotes (i.e. add a rule to your CSS file that says that all blockquotes should use italic).

Colors

In lecture, we discussed several ways to set the color of elements. You can use one of the 17 colors that can be specified by keyword (i.e. red, blue, green, and their friends), or you can specify the red, green, and blue content (RGB) of your desired color.

For this part of the lab, we will set colors using RGB content.

  1. Choose your favorite color. If you don't have a favorite color, use Tony's (a dark orange color).
  2. Take Tony's journal page, journal.html and style sheet, journal.css, and:
    • change the largest headings (h1) to be your favorite color,
    • change the next largest headings (h2) to be a lighter hue of your favorite color,
    • change the text in paragraphs to be a darker hue of your favorite color,
    • add a dotted bottom border to the headings. 
To find the RGB of your favorite colors, you can use the color picker in the Paint program, experiment by adjusting RGB values in the CSS file, or use the color chart webpages suggested by the book.
