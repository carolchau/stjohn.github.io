Blog Entries

We will use the blog developed in Chapter 9 of the book:

http://headfirstlabs.com/books/hfjs/ch09/youcube/youcube4.html

The initial tasks are:

  • To familiarize yourself with the code, your first task is to add three more blog entries to the existing blog.
  • Next, modify the Blog object to also include the author of the blog. That is, change the constructor to take a third argument (the name of the author) and set it inside the constructor (in a similar fashion to that for the entry and date. Change the existing blog entries to have authors (give at least three different authors).
  • Change the existing blog entries to include the author of each entry.
  • Change the showBlog() function to also show the author for each entry

Sorting by Date and Author

The existing blog sorts the entry by date. Since we now store the author, we'll give the user the choice of sorting by date or by author's name.

  • Add a button that allows the user to sort by date (hint: use the existing sort function).
  • Add a button and associated function that allows the user to sort by author
    Hint: this is very similar to what you did for dates, except to compare strings use: 
    			return blog1.author > blog2.author
