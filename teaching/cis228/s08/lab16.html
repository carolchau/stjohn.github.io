Today, we'll continue working with storing information in JavaScript (using variables and cookies) and will also use some simple functions to interact with the browser. These include using timers and reacting appropriately when the browser window has been resized.

Your Pet

The book develops the iRock in the first three chapters. Now, it's your turn. We will use the iRock files at:

http://headfirstlabs.com/books/hfjs/ch03.php
Modify the iRock files to create your own interactive pet.
  1. Find two images for you pet: a happy one and a sleeping one
  2. Modify the iRock example (the last file, iRock8.html is a good one to use) to:
    1. use your images in place of the iRock,
    2. have your pet "fall asleep" after 30 seconds of no interaction,
    3. when you click on the screen, your pet should wake up (i.e. display the happy image) and thank you for waking him up.

Using Variables

Next, let's write a simple program that uses variables. Write a webpage that uses JavaScript to:

  1. first, ask the user for a number and
  2. then tells the user the square of that number

For example, if the user enters 2, you say 4; if the user enters 3, you say 9.

Hint: do this in parts, making sure each part works before going onto the next.
