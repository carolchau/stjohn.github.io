More on Layouts

We'll style the journal.html from Chapter 9. The goal is to add navigation on the side (like you see in a blog).  

  1. Create a sidebar with links to the entries in his journal. For example, the first link should be to the entry for August 20, the next link for the entry on July 14. Use the id attribute to create anchors in the page (see p 153 for a review).
  2. The sidebar should be to the right, below the title of the page. Use any page layout or rules you wish to place it there.
  3. The main part of the page and the sidebar should each have borders around them and different background colors.
