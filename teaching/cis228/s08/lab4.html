General Information & Getting Started

See Lab 1 for more details on getting started and the rules for the lab.

Today's focus is using connections to the internet. We will talk briefly about creating your own domain and all that entails. We will spend the majority of the lab on linking to URLs, absolute paths, and fun uses of the <a> element like popping up new windows, adding titles, and creating destinations.

Favorite Places

To start off today, make a simple page in your AllAboutMe site, called places.html. The page should have:
  1. A title, that includes the name of a place you would like to visit.
  2. A short description of a place that you would like to visit and why.
  3. A link to another page (ie use an URL) that describes the place you'd like to visit.
  4. If time, several images of the place. Put those in your images folder, and use relative paths in the links to them.

Starbuzz and Caffeine Buzz

Before starting the Segway exercises, you will need to download the files provided on the book's webpage to a floppy drive or USB keychain.  If you haven't already done so, the book's website is:
http://headfirstlabs.com/books/hfhtml/index.php
The original page for Starbuzz Coffee is provided.  We'll modify it to include some of the new features we've discussed in class:
  1. Add in an absolute link to the Caffeine Buzz website.
  2. Add a title to that link, that explains what it does.
  3. Modify the <a> element for the Caffeine Buzz link to pop up a new window.

All-About-Me Website

The final activity for today's lab we will add more to the website with information about you. 
  • Add titles to all the links on your default page, describing their content.
  • Link your places page to the default page and have it pop up in a new window.

Submitting Your Lab

Show your webpages to Professor St. John, before leaving the lab today.  

Logging Out

Before leaving the lab, you need to log out and shut down your computer. Remember to pick up any files you sent to the printer and to take any personal belongings. It is difficult to get back into the lab later (since it's locked when not in use), so, it's worth making sure you haven't forgotten anything.
