Your Pet, Revisited

The Document Object Model (DOM) provides a way to access and change parts of your webpage, even if those sections lack IDs. In lecture, we discussed improvements to the Stick Figure Adventure that show messages directly on the page (instead of using alert()'s. In today's lab, we will make similar improvements to your pet page.

Alter your pet page from Lab 16 to have:

  • A message that appear below the picture. The initial message should be your greeting.
  • Instead of using an alert() to say thank you when woken (ie clicked on), change the message on your page to include the thank you.
  • When your pet falls asleep, change the message on your page to reflect his sleepy state.
In your pet page, the change in images was accomplished with an interval timer (ie setTimeout()). One place you will need to add more code is in this timer. If it gets too confusing and long, create a function that contains the resetting of the message and image, and just use that function inside the interval timer.
