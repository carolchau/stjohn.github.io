Today's focus is using connections to the internet.  We will talk briefly about creating your own domain and all that entails.  We will spend the majority of the lab on linking to URLs, absolute paths, and fun uses of the <a> element like popping up new windows, adding titles, and creating destinations.

Divs and Spans

Before starting the lounge exercises, you will need to download the files provided on the book's webpage to a floppy drive or USB keychain.  If you haven't already done so, the book's website is:
http://headfirstlabs.com/books/hfhtml/index.php
We'll work today with the lounge.html from Chapter 11:
  1. Add a <div> element to the lounge.html and lounge.css for the footer (the copyright information at the bottom of the page).  Now, using style rules (only for the footer!), make the text centered, small (50%), and add a generous top margin.
  2. Add <span> elements for the CDs and artists at the end of the page, and style them separately (italics for CDs, bold font for artists).
  3. Add to the style sheet, rules that change the colors of links when unvisited, visited, and hovering.  Use any colors (the book suggests #007e7e, #333333, and #0d5353).
  4. Move the elixirs section of the HTML file directly below the logo, and add the following rule to the #elixirs selection:  float: right; What happens?

Flow and Layouts

Next, we're going to look at some layout.  We'll work with the starbuzz.html file in Chapter 12 (the coffee example, revisited):  We'll focus on the CSS file (since the <div> element has already been added to the HTML file):
  1. In the HTML file, move the sidebar logical section to where you would like it to "float" (ie right below the logo and intro).
  2. In CSS, set the width of the sidebar logical section.
  3. In CSS, add the rule to "float" the sidebar to the right hand side of the page.

All-About-Me Website

Take your favorite foods page and add a sidebar that gives a short description of where you can find those foods-- either links to restaurants or to recipe books if home-made.  Make sure you place the sidebar section in the correct place to "float" (ie below the opening heading, but above your other text).  Note: to float a section, the section must have its width set (otherwise it defaults to the width of the whole page and the float will appear to do nothing).
