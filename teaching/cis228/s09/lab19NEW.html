Stick Figure Adventure

First, we'll finish up a program from Chapter 4: Stick Figure Adventure

http://headfirstlabs.com/books/hfjs/ch03.php

There are two paths that currently end as "To Be Continued." Choose one of those paths and extend it with at least two more scenes. For each of your new scene, find an image to represent it and store it in the directory of images. You should have at least one decision in your new part of your adventure.

Hint: You will find it easiest to use images that are already in the PNG format and name them after the scene they will appear in (e.g. scene10.png). If you have an image that's not in PNG, you can convert it via several online programs. Here's a partial list:

Loops

In lecture, we discussed two different types of loops: for loops and while loops. For each of the programs below, you can use either type of loop you like:

  1. Write a program that will display 10 different hues of blue. Note that in class we discussed a program that displayed 10 different hues of red.
  2. Write a guessing game program that asks the user to guess your secret number. If they enter a number that's too low, tell them so, and prompt them for another guess. Similarly, if they enter a number that's too high, tell them so, and prompt them for another guess. When they guess the correct number, congratualate them.
