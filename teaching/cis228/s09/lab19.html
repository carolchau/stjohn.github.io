Timely Greetings

Write a JavaScript program that will greet the user with "Good Morning" if it is before 11 in the morning, or "Good Day" after 11. A good place to start would be to modify an early iRock file that greets the user.

To find out the current hour, use the following piece of code: 

    var currentDate = new Date();
    var hour = currentDate.getHours();

Stick Figure Adventure

First, we'll finish up a program from Chapter 4: Stick Figure Adventure

http://headfirstlabs.com/books/hfjs/ch03.php

Starting with sfa1.html, there are two paths that currently end as "To Be Continued." Choose one of those paths and extend it with at least two more scenes. For each of your new scene, find an image to represent it and store it in the directory of images. You should have at least one decision in your new part of your adventure.

Hint: You will find it easiest to use images that are already in the PNG format and name them after the scene they will appear in (e.g. scene10.png). If you have an image that's not in PNG, you can convert it via several online programs. Here's a partial list:

  • http://www.piccyfix.com/
  • http://snipshot.com/
