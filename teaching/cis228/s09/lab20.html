Guessing Game

Write a guessing game program that asks the user to guess your secret number:

  • If they enter a number that's too low, tell them so, and prompt them for another guess.
  • Similarly, if they enter a number that's too high, tell them so, and prompt them for another guess.
  • When they guess the correct number, congratualate them and end the game.

A good place to start is by modifying the "guess a password" program from the while loops example.

Color Table

Write a program that will display 10 different hues of blue. Note that in class we discussed a program that displayed 10 different hues of red.
