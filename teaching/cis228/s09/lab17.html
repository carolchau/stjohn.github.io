In today's lab, we will work more with variables and also learn ways to communicate with the browser from JavaScript.

Using Variables

First, let's write a simple program that uses variables. Write a webpage that uses JavaScript to:

  1. first, ask the user for a number and
  2. then tells the user the square of that number

For example, if the user enters 2, you say 4; if the user enters 3, you say 9.

Hint: do this in parts, making sure each part works before going onto the next.

Communicating with the Browser

Built in to JavaScript are ways to communicate with the browser. Following the modified iRock example, change the annoying window example to resize the window whenever the browser window resizes. That is, if the user makes the browser window bigger, the annoying window should also get bigger with the goal that it always covers at least three quarters of the screen.
