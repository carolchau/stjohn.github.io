Mandango Seats

First, we'll work with the Mandango program from Chapter 6:

http://headfirstlabs.com/books/hfjs/ch06.php

Add a button to the webpage, "Clear Seats", which resets all the seats to empty. To do this, you will also need to write a new (very short) function that is called when the button is clicked.

Note: an easy way to make sure your changes are shown on the webpage is to call the initSeats() function after you have reset the values of the array.

Functions with Parameters

First, create a new webpage that will display different messages when different sections of the page have been clicked, using the displayMessage() function from functions.html. On your new page, you should have at least three elements that display messages when clicked. None of your three elements may be form buttons; instead, use images, divs, tables, etc. Each of the three elements should display a different message.
