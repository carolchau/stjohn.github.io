Loops

In lecture, we discussed two different types of loops: for loops and while loops.  For each of the programs below, you can use either type of loop you like:

  1. Modify the temperature table program to display the temperatures, ranging from -100 to 100 degrees celsius, counting by tens (ie -100, -90, -80,... , 80, 90, 100).
  2. Write a program that will display 10 different hues of blue.   Note that in class we discussed a program that displayed 10 different hues of red.
  3. Write a guessing game program that asks the user to guess your secret number.  If they enter a number that's too low, tell them so, and prompt them for another guess.  Similarly, if they enter a number that's too high, tell them so, and prompt them for another guess.  When they guess the correct number, congratualate them.
