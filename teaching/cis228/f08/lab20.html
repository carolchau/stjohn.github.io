Airplane Reservations

The first part of today's lab is to change the airplane seat reservation program to allow the user to "unselect" a seat. With the current version, once a seat is selected, it can't be undone.

Change the program so that if an occupied seat (i.e. one that says "Full") is clicked, it then returns to being unoccupied (and displays "Coach"). Note that you only need to change one block of code to accomplish this.

Some useful images to have in your directory are:

  • coach icon and
  • full icon.

Tic Tac Toe

In lecture, we discussed how to set up the Tic-Tac-Toe game using an array. The example in class allowed the users to play, but didn't check for a winner. Your assignment is to fill in the function, checkForWinner(). The function should return true if there's a winner and false otherwise. A player wins at Tic-Tac-Toe if they have three in a row, in a column, or diagonally.

For Tic Tac Toe, you will need the following files:

  • index.html
  • O.jpg
  • X.jpg
  • empty.jpg

The easiest way to do this program is to first write the JavaScript to see if the moves in the first row all belong to the same player (and are not equal to "empty"). After that, then write the test for the other rows, then the columns, and then the diagonals.
