General Information & Getting Started

New topics for this lab include the <div> element and the <span> element which are used to divide the pages into smaller parts that can be styled individually.

divs

Before starting the exercises, you will need to download the files provided on the book's webpage to a floppy drive or USB keychain.  If you haven't already done so, the book's website is:
http://headfirstlabs.com/books/hfhtml/index.php
We'll start today with the lounge.html from Chapter 11, adding a menu and footer to the page:
  1. Add a <div> element to the lounge.html and lounge.css:
    • the identifier (id) should be elixirs and contain the elixir menu,
    • the menu should have width 200 pixels, a thin solid border, centered text, and 20 pixels of padding on all sides but the top.
    • further, use the background image, placing it in the upper left hand corner and having it repeat exactly once
  2. Next, using descendant selectors, change the color of the headings inside the elixirs section only:
    • the <h1> headings should be black, and
    • the <h2> headings should be red
  3. Add a <div> element to the lounge.html and lounge.css for the footer (the copyright information at the bottom of the page).  Now, using style rules (only for the footer!), make the text centered, small (50%), and add a generous top margin.

spans

<span> elements are used to style in-line elements. We'll first work with the CDs and artists at the bottom of the lounge page:
  1. Add <span> elements for the CDs and artists at the end of the page, and style them separately (italics for CDs, bold font for artists).
  2. Move the elixirs section of the HTML file directly below the logo, and add the following rule to the #elixirs selection:  float: right; What happens?

All-About-Me Website

Make a new page, based on your favorite places page, called travel.html, and an associated style file, travel.css.  This page should have at least two places, consisting of a heading, paragraph, and image.  Using the <div> element, divide each place into its own logical section.  Place a border around each section and change the background color and text color to complement the image chosen for that section.
