General Information & Getting Started

Today's focus is using connections to the internet.  We will talk briefly about creating your own domain and all that entails.  We will spend the majority of the lab on linking to URLs, absolute paths, and fun uses of the <a> element like popping up new windows, adding titles, and creating destinations.

Forms

Following the Starbuzz example from the textbook (Chapter 14). Create a form for ordering sandwiches
from the "228 Sandwich Shop".  Your form should have the following choices (the name used by the web application is included in parenthesis): 
  • first and last names (firstname and lastname)
  • type of bread (bread)
  • toasted or not (cookingInst)
  • choice of filling (filling)
  • extras (extras[]),
  • and a text area for comments and special instructions (comments)
For the first three, you should allow only one answer (ie radio buttons or pull-down menu), while
for extras (ie mayo, mustard, tomato, etc), multiple answers are expected (ie checkboxes).

The web application for processing the form is located at:

http://comet.lehman.cuny.edu/stjohn/teaching/cis228/sandwich/order.php

Adding Style

Take the basic form that you created above, and add style.  Layout the form using a table.
Then add at least three CSS rules that make the form nicer to look at.  Some suggestions include:
adding color, changing the borders in the table, and making the heading stand out via borders
and size.
