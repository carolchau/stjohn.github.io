We'll work today with the starbuzz.html from Chapter 12 (the coffee example, revisited). We'll focus on the CSS file (since the <div> element has already been added to the HTML file):

  1. In the HTML file, move the sidebar logical section to where you would like it to "float" (ie right below the logo and intro).
  2. In CSS, set the width of the sidebar logical section.
  3. In CSS, add the rule to "float" the sidebar to the right hand side of the page.

More on Layouts

We'll style the journal.html from Chapter 9. The goal is to add navigation on the side (like you see in a blog).

  1. Create a sidebar with links to the entries in his journal. For example, the first link should be to the entry for August 20, the next link for the entry on July 14. Use the id attribute to create anchors in the page (see p 153 for a review).
  2. The sidebar should be to the right, below the title of the page. Use any page layout or rules you wish to place it there.
  3. The main part of the page and the sidebar should each have borders around them and different background colors.

All-About-Me Website

Take your favorite foods page and add a sidebar that gives a short description of where you can find those foods-- either links to restaurants or to recipe books if home-made. Make sure you place the sidebar section in the correct place to "float" (ie below the opening heading, but above your other text).

Also, add style commands that alter how links are displayed in your page. You can just change the color, as in the lounge page above, or be more creative.

Note: to float a section, the section must have its width set (otherwise it defaults to the width of the whole page and the float will appear to do nothing).
