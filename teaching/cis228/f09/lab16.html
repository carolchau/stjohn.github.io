Using Variables and Constants

In the iRock examples, we used a variable (userName) to store the name of the user and use it in a greeting. Variables can also be used to store information entered into forms. We will add additional variables to the donut shop example from the book, to allow more donuts and bagels to be purchased.

We build a bigger donut shop that allows the user to order other kinds of donuts. (You will need to download the files from Chapter 2 to complete this sections of the lab. Start with the file donuts5.html. Note that const does not work on Internet Explorer. If you are using IE, make sure you download the alternative version of this page.)

Add to the donut page:

  • another kind of donuts that can be ordered (at 50 cents a donut),
  • also allow the user to order bagels (at 75 cents a bagel),
  • following the format for cake and glazed donuts, include these two new options in the final total for the order.
