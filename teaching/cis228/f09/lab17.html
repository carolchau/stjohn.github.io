Changing Style

In the annoying window example, a window repeatedly pops up after 3 seconds. Make the following changes to the program:

  • Make the window reappear after 1 second.
  • Add another "button" that when pushed, makes the window twice as wide.
    (Hint: the width of the window can be set, using style.width.)
  • Add another "button" that when pushed, turns the background color of the window red.

iRock Friend

Working with iRock6.html from Chapter 3, create a new file where iRock has a friend. You can create a new character (ie find 2 images on the web) or reuse the images of iRock. When the page first loads, the user should see the two friends on the page. The old friend (ie iRock) should start out sad, but become happy when clicked on. The new friend should do the opposite: it should start out happy but become sad when clicked on.
