Back to Style

Before starting the lounge exercises, you will need to download the files provided on the book's webpage to a floppy drive or USB keychain. If you haven't already done so, the book's website is:
http://headfirstlabs.com/books/hfhtml/index.php
We're going to work with the lounge.html file in Chapter 8 (the lounge example, revisited).
  1. Add the style described in the book lounge.html to change:
    • the font in paragraphs to maroon
    • make headings use sans-serif fonts in gray
    • add underlining (bottom-border) to h1 headings
  2. Now, move your style rules to a new page lounge.css, and link it to lounge.html.
  3. Use the same set of style rules (ie link to lounge.css) for directions.html, and exilir.html.

Classes

The last part of Chapter 8 covers classes in CSS.
  1. Add classes to the elixir.html and lounge.css to make:
    • the paragraph about green tea have green text.
    • the paragraph about raspberry elixir have blue text.
    • the paragraph about blueberry elixir have purple text.
  2. Do the exercise on p 326.
  3. Validate your lounge.css file using the CSS validator:
    http://jigsaw.w3.org/css-validator
