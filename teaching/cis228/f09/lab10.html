Spacing, Borders, and Margins

Before starting the lounge exercises, you will need to download the files provided on the book's webpage to a floppy drive or USB keychain. If you haven't already done so, the book's website is:

http://headfirstlabs.com/books/hfhtml/index.php

We'll work with the lounge.css file in Chapter 10 (the lounge example, revisited):

  1. In the lounge folder for Chapter 10, create a file lounge.css with:

    body {
       font-size: small;
       font-family: Verdana, Geneva, Arial, sans-serif;
    }
    h1,h2 {
       color: #007e7e;
    }
    h1 {
       font-size: 150%;
    }
    h2 {
       font-size: 130%;
    }
  2. The link already exists in lounge.html, so, when you view the file you should see the effect of the above rules.
  3. Our goal is to match the completed journal page.  To do this, alter the style for the guarantee paragraph:
    1. Add space around text (hint: padding).
    2. Increase all the margins.
    3. Add in the background image, images/background.gif and place a single copy of it in the upper left hand corner.
    4. Add a dashed white border around the text.
    5. Adjust the left padding and right margins to match the completed page.
