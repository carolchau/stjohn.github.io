Getting Started with JavaScript

For your first JavaScript program, we're going to add a "Hello World" dialog box to an existing page.  So choose your favorite page from your AllAboutMe website, and add an alert() that says "Hello World!" when the page loads.  Note this follows the format of simpleJS.html almost exactly.

Clicking Event

In lecture, we had a page that reacted when the mouse was clicked (variables.html).  This was accomplished using the onclick() function.  If the onclick() method is put in the <body> element, then it applies to the whole page.  What happens if it is put in another element?  Add an image to the
variables.html page and move the onclick() method call to the <img> element.  What happens?

Add a second image to the page, and have it do something different (ie displays a different message, brings up a different kind of dialog box, etc) when the user clicks on that image.
