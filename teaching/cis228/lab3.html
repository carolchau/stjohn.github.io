Today, we're going to go from a page idea to an outline to the actual webpage.  Along the way, we'll talk about some new elements for quoting and formatting text, and discuss nesting of elements.

Segway Journal

Before starting the Segway exercises, you will need to download the files provided on the book's webpage to a floppy drive or USB keychain.  If you haven't already done so, the book's website is:
http://headfirstlabs.com/books/hfhtml/index.php
The basic journal page for Tony's Segway journey is already in the files provided above.  As a review, we'll discuss in class the elements on his page.  Next, modify his page to take advantage of the new structure elements we've discussed:
  1. Add <q> element to properly quote his inline quotes.
  2. Use a <blockquote> and <br> elements to make his Burma Shave quote appear correctly.
  3. Make an ordered list of the cities he's visited along his journey
  4. Make an unordered list of the things he brought with him on the trip.

All-About-Me Website

The final activity for today's lab we will add more to the website with information about you. 
  • If you didn't do so last class, make a folder, called allAboutMe (the root folder) to hold your website and put your webpage from last time in the folder, renamed as index.html (why should you name it index.html?)
  • Add your favorite quote (either inline or block) to your default page.
  • Create a new page, titled "Fall 2007 Class Schedule".  On that page make an unordered list of your courses for this semester.
  • Link your new course page to your default page.
