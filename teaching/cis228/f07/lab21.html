Today's focus is using cookies to store information after the browser has exited or has reloaded the page.  In lecture, we also discussed how dates and times are stored in JavaScript and the use of innerHTML() to set values on a webpage.

Timing Visits

In the cookie.html file, we displayed the number of previous visits to the page.  Add an additional <span> element that says the last time you visited the page.  (Hint:  add an additional cookie to hold the date when you last visited).

Using Dates

Write a javaScript function that displays the current date on the top of the webpage.  This program does not use cookies but does use the other topics discussed in lecture: the Date object and innerHTML() function.  To break this program into steps:
  1. Create (or reuse from a previous lab) an HTML file with a heading section on the top.  The heading should be a unique div with id of "heading".
  2. Write a (very short) JavaScript function that gets the current date (hint:  see setCookie() function in cookie.html) and then sets the #heading div to that value (hint: see updateCount() function in cookie. html).
  3. Your JavaScript function should be called whenever the page is reloaded (hint: <body onload="...">).
