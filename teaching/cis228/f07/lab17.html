Using Variables

First, let's write a simple program that uses variables.  Write a webpage that uses JavaScript to
  1. first, ask the user for a number and
  2. then tells the user the square of that number
For example, if the user enters 2, you say 4; if the user enters 3, you say 9.

Arithmetic in JavaScript

In lecture, we wrote a calculator for determining the letter grade given the numerical score.  In lab, we'll write a calculator that takes a decimal number and outputs its hexadecimal code.  To make things simpler,
we will only convert numbers that are < 16.  The basic outline of the program is:
  1. Prompt the user for a number.
  2. If the number is >= 16, display a message saying the number is too large for our calculator.
  3. If the number <=15, print out the single digit hexadecimal for that number
Since there's lots of steps to this, let's break it up into subproblems, detailed in the sections that follow:
  • Print and Display:  To start, first make a program that prompts the user for a number and just prints out that same number to the screen (this is to check that your variables are set up right).
  • Numbers That Are Too Big:  Next, add in a test if the number is bigger than 15.  If so, print out a message saying so.
  • Small Numbers:  if the number is <=9, then you can just print out the number (since numbers that small are the same in hexadecimal or decimal).  Add in a test for this, and test to make sure it works.
  • Converting the Rest:  Next, if the number is 10, print out "A" (add a test for this).  If the number is 11, print out "B" (add a test for this).  Repeat with the numbers upto 15.  In the end, you'll have several tests, one for each of the numbers between 10 and 15.  Test to make sure this works.
In a future lab, we'll use this work to convert larger hexadecimal numbers.
