Today's focus is using connections to the internet.  We will talk briefly about creating your own domain and all that entails.  We will spend the majority of the lab on linking to URLs, absolute paths, and fun uses of the <a> element like popping up new windows, adding titles, and creating destinations.

Layouts

Before starting the lounge exercises, you will need to download the files provided on the book's webpage to a floppy drive or USB keychain.  If you haven't already done so, the book's website is:
http://headfirstlabs.com/books/hfhtml/index.php
We'll finish today with the journal.html from Chapter 9.  The overall goal is to style Tony's Segway page with navigation on the side (like you see in a blog).  
  1. Create a sidebar with links to the entries in his journal.  For example, the first link should be to the entry for August 20, the next link for the entry on July 14.  Use the id attribute to create anchors in the page (see p 153 for a review).
  2. The sidebar should be to the right, below the title of the page.  Use any page layout or rules you wish to place it there.
  3. The main part of the page and the sidebar should each have borders around them and different background colors.

All-About-Me Website

Take your schedule page from before, and add a fixed element with a link to the Lehman College course schedules.  This element should stay in the same place, no matter if the text in the main window is scrolled up and down.  See the "free coffee coupon" example in Chapter 12 as a guideline.  Your page must have:
  1. A fixed element with a link to the Lehman College course schedule.  Your link can be words or an image.
  2. If the link is an image, remove any borders (ie use pseudoclasses).  If the link is words, color the section differently, so it stands out from the background of the main section of your page.
