General Information & Getting Started

See Lab 1 for more details on getting started and the rules for the lab.

Today's focus is using connections to the internet.  We will talk briefly about creating your own domain and all that entails.  We will spend the majority of the lab on linking to URLs, absolute paths, and fun uses of the <a> element like popping up new windows, adding titles, and creating destinations.

Functions

Use the displayMessage() function from functions.html to create a new page.  On your new page, you should have at least three elements that display messages when clicked.  None of your three elements may be form buttons, instead, use images, divs, tables, etc.  Each of the three elements should display a different message.

Events

In lecture, we discussed how to validate if fields of a form were empty (see formJS.html).  For this part of lab, we will extend that program to check if the state abbreviation and zip codes are of the proper length. Doing this program breaks into several parts:
  1. Write a function, validateState(), that tests if the input if of length 2.
  2. Write a function, validateZip(), that tests if the input is a number and if of length 5.
  3. Connect each of these functions to the appropriate form fields, when the focus leaves these fields (hint: onblur).
