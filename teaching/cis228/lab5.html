General Information & Getting Started

Today's focus is using connections to the internet.  We will talk briefly about creating your own domain and all that entails.  We will spend the majority of the lab on linking to URLs, absolute paths, and fun uses of the <a> element like popping up new windows, adding titles, and creating destinations.

myPod Website

Before starting the Segway exercises, you will need to download the files provided on the book's webpage to a floppy drive or USB keychain.  If you haven't already done so, the book's website is:
http://headfirstlabs.com/books/hfhtml/index.php
We're going to work with the index.html file in Chapter 5 (for the myPod site). 
  1. Add in the image, seattle.jpg, provided in the Chapter 5 folder.  What happens?
  2. Since it's huge, resize the image to 50% using Paint (under the accessories menu on Windows), iPhoto (on the Mac), or Adobe Acrobat Elements if you own a copy.
  3. Add in the rest of the images to the webpage, grouping the Seattle and England pictures together.
  4. Make a thumbnails directory, and resize the pictures to 12.5% and save them there.
  5. Redo the page with thumbnails, instead of full size pictures.  Include <alt> tags in each <img> element.
  6. Next, make the thumbnails into links to pages containing the full size pictures.

All-About-Me Website
