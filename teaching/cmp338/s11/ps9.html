This problem set covers searching graphs. See the class webpage for references.

In this assignment, you will find paths between points and calculate the size of components in a random graph. The necessary files are available on Blackboard.
  1. Isolated Vertices, Paths, and Components (60 points)

    The file RandomGraphs.java (available on Blackboard) uses an adjacency matrix to represent (Erdos-Renyi) random graphs.

    Write the following methods for the random graphs:

    • int isolatedVertices() returns the number of isolated vertices in the graph, i.e. those nodes that have no neighbors
    • boolean isPath(int source, int target) returns true if there is path between source and target. Otherwise, returns false.
    • boolean connected() returns true if the graph is connected (i.e. there's a path between every two nodes) and false otherwise.
    • int numComponents() returns the number of connected components in the graph. This number is bounded between 1 (the completely connected graph) and SIZE of the graph (a graph consisting of all isolated vertices).

    Submit your program as a .java file via Blackboard.

  2. Test Cases (20 points)

    For graphs with 20 vertices and edge probability = 1/10, 1/9, 1/8, 1/7, 1/6, 1/5, 1/4, 1/3, 1/2, 1/1, report the following:

    • the fraction of isolated vertices (i.e. the number of isolated vertices divided by the total number of vertices, 20)
    • the number of components in the graph
    • if there is a path from node 1 to 2
    • if the graph is connected
    • the number of components in the graph

    Run the test 10 times and make a table with your average fraction of isolated vertices and average number of comoponents as well as the standard deviation. For the last three, report the fraction of time the condition held (i.e. 9/10 of graphs were connected).

    Submit your table via Blackboard.

  3. Analysis (20 points)

    Using your data above, answer the following questions:

    • How does the fraction of isolated vertices change as the edge probability increases?
    • How does the number of components as the edge probability increases?
    • How does connectivity of the graph change as the edge probability increases?
    • Are the change in connectivity and number of isolated vertices related? Why or why not?
    • Are the change in connectivity and existence of a path related? Why or why not?

    Submit your answers as a text file via Blackboard.
