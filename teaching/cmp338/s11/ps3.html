Problem Set 3

CMP 338: Data Structures and Algorithms
Lehman College, City University of New York
Spring 2011

This problem set covers two new sorts covering in class: quicksort and radix sort. See the class webpage for references.

  1. QuickSort (30 points)

    Using the SortingStrings.java as your driver program, implement the in-place QuickSort algorithm (ie the one at the end of the "Complex version" section on the wikipedia page).

    Instead of an array of integers, have your quicksorts sort an array of Strings (that is, String[] A). Note that to compare strings in Java, you should use string1.compareTo(string2) where string1 and string2 are the names of your string variables.

    Submit the java file containing the quicksort and a main that calls it to sort a list.

  2. RadixSort (30 points)

    Implement a radix sort to sort arrays of strings of length 6. Use the SortingString.java file as your driver. Note that in that file, there is an implementation of InsertionSort that can be used as the underlying stable sort for your radix sort.

    Submit the java file containing the radix sort and a main that calls it to sort a list. Note that you can include both sorts in the same file.

  3. Test Cases (20 points)

    For both sorting algorithms, calculate how long each algorithm takes to sort the following lists:

    • a list of 100 random 6-character words
    • a list of 200 random 6-character words
    • a list of 300 random 6-character words
    • a list of 400 random 6-character words
    • a list of 500 random 6-character words

    Run each test 10 times and make a table with your average as well as the standard deviation for each algorithm. Submit your table via Blackboard.

  4. Analysis (20 points)

    Using the table you created, answer the following questions:

    • From your data, which sort did the best overall? Why?
    • We know that asymptotic running time of radix sort is O(n) while the asymptotic running time of the quick sort is O(n lg n). Did your data support this? Why or why not?

    Submit your answer via Blackboard.
