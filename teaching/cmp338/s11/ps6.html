Lehman College, City University of New York

This problem set covers stacks. See the class webpage for references.

This assignment focuses on using stacks for parsing and simple calculations. The necessary files are available on Blackboard.
  1. Adding Operators (30 points)

    Modify the postfix program (available on Blackboard) to allow the user to enter the additional operators:

    • subtraction (-),
    • division (/),
    • increment (++),
    • decrement (--)

    Submit your program as a .java file via Blackboard.

  2. Analysis (20 points)

    What is the order of growth of the worst case time needed to process an arithmetic expression? Justify your answer.

    Submit your answer via Blackboard as a text or word file.

  3. Shunting Algorithm: From Infix to Postfix (50 points)

    Implement Dijkstra's Shunting Algorithm (see webpage for algorithms) that converts infix notation to postfix. Your program should take as input a string in infix notation, use the Shunting Algorithm to convert the input to postfix notation, and then use your program above to calculate the answer.

    For example, if the user enters 12 * (4 + 5) / 6, you should convert it to 12 4 5 + * 6 / and then apply the calculator code (from the first part of the homework) to yield 18. See http://www.chris-j.co.uk/parsing.php for excellent examples.

    Submit your answer as a .java file via Blackboard.
