  • The programming language for the course is Not Quite C (NQC), written by Dave Baum, and maintained by John Hansen. Some useful NQC documents include:
    • NQC Programmer's Reference Sheet by Matthew Bates:
      2 pages with every NQC command and construct, very briefly described.
    • NQC Programmer's Guide by Dave Baum:
      62 page guide to programming in NQC.
    • Some BricxCC tutorials by the original authors (towards the bottom of the page).
    • The Bricx Command Center software for Windows.
