Suggested Problems

The following questions are suggested to prepare for the quiz.  They are based on the above reading and should be attempted before starting the quiz.  You do not need to turn in written answers to the questions below.  Instead, you should log on to Blackboard and take the quiz during the time allotted.

Undergraduate Questions

The following questions are for those taking the course for undergraduate credit:
  1. What is a path?
  2. What is the difference between a user and a group in unix?
  3. List 3 different shell programs available on unix.
  4. What does the command tail do?
  5. What does the command more do?
  6. How would you count the number of words in a file?
  7. What does gzip do?  How do you undo this command?
  8. What does crontab do?

Graduate Questions

In addition to the undergraduate questions, those taking the course for graduate question should also do the following questions:

  1. In unix shell, how would count the number of lines in a file that contain the substring "AAA?"
  2. What is the difference between telnet and ftp?
  3. How do you list all the processes on the system that belong to root? (Note: there's several ways to do this).
  4. What does the following command do:  "sort -k 2 exam_scores | head -n 10"?
  5. What does the following command do:  "ls *.big | grep "not needed" | rm -rf" ?
