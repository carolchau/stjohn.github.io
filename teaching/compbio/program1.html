General Information

The following programs should be written in the perl programming language and turned in via the Dropbox option in Blackboard.  When turning in your programs, you should submit plain text files and call the programs youlastnameProgramX.pl where X is the program number.  

To obtain full credit for your program, the initial comment block at the beginning of the file should include your name, your preferred email, and a short description of the program.  You should comment any non-obvious sections of codes.  Use your own words to describe the program in comments.  Remember that copying others programs is considered cheating.  Submitted programs will be randomly compared to look for similarity and cheating.

Undergraduate Programs

The following program are for those taking the course for undergraduate credit:
  • UG1:  Write a perl program that prompts the user for an input sequence, reads in the sequence from standard input, and if it's a DNA sequence transcribes it to RNA.  If the input is an RNA sequence, it reverses transcribes the sequence from RNA to DNA.  The output of your program should be a sequence displayed to standard out (ie the screen).
  • UG2:  Write a perl program that prompts the user for a DNA sequence, reads in the sequence from standard input, and scans the string for a stop codon (see Figure 2-3, p 26 for possible stop codons).  Your program should output the number of stop codons seen.

Graduate Program

In addition to the undergraduate programs, those taking the course for graduate question should also do the following program:

  • G1:  If your first name starts with A-M, do program G1, otherwise do G2:
    Write a perl program that prompts the user for a DNA sequence, reads in the sequence from standard input.  Your program should then calculate how CG-rich the sequence is.  That is, you should report the percentage of the sequence that contains C's and G's.  This is an important aspect of a sequence since the CG content of regions often indicate whether they're coding or non-coding regions.
  • G2:  If your first name starts with A-M, do program G1, otherwise do G2:
    Write a perl program that prompts the user for a protein sequence, reads in the sequence from standard input.  Your program should then calculate the percentage of hydrophobic (water-fearing) amino acids it contains.  That is, you should report the percentage of the amino acids in the sequence that are hydrophobic (see any molecular biology text book such as this one for a list of the water-fearing amino acids).  This is an important aspect of a sequence since the response to water affects both the folding and function.
