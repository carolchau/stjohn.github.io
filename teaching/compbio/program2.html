General Information

The following programs should be written in the perl programming language and turned in via the Dropbox option in Blackboard.  When turning in your programs, you should submit plain text files and call the programs youlastnameProgramX.pl where X is the program number.  

To obtain full credit for your program, the initial comment block at the beginning of the file should include your name, your preferred email, and a short description of the program.  You should comment any non-obvious sections of codes.  Use your own words to describe the program in comments.  Remember that copying others programs is considered cheating.  Submitted programs will be randomly compared to look for similarity and cheating.

Undergraduate Programs

The following program are for those taking the course for undergraduate credit:
  • UG3:  Write a subroutine that given a codon, returns a list of all the amino acids that may result from a single mutation in the codon.   For example, the codon UUU (which codes for Phe) can mutate at the first position to give the codons AUU, CUU, GUU.  Similarly, if UUU has a single mutation in the second or third positions, then this gives the codons UAU, UCU, UTU, UUA, UUC, and UUT.  These 9 codons code for Phe (UUC), Ser (UCU), Tyr (UAU), Cys (UGU), Leu (CUU, UUA, UUG), Ile (AUU), and Val (GUU).  So, your subroutine should return: Phe, Ser, Tyr, Cys, Leu, Ile, and Val.  Your program should prompt the user for a codon, call the subroutine and print out its results.
    (You might find the library of perl routines from the recommended text, Beginning Perl for Bioinformatics, useful for the program). 
  • UG4:  Some codons are more likely than others to occur in DNA.  Write a subroutine that, given an amino acid, will return the probability the number of codons that code for that amino acid.  Your program should prompt the user for a amino acid, call the subroutine, and print out its results.
    (You might find the library of perl routines from the recommended text, Beginning Perl for Bioinformatics, useful for the program). 

Graduate Program

In addition to the undergraduate programs, those taking the course for graduate question should also do the following program:

  • G3:  Write a perl program that randomly mutates the amino acids in a protein, but resticts the possibilities to those that can occur due to a single mutation in the original codons.  You may want to use the subroutines you wrote for the above undergraduate programs for this program.
