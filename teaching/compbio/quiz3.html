Suggested Problems

The following questions are suggested to prepare for the quiz.  They are based on the above reading and should be attempted before starting the quiz.  You do not need to turn in written answers to the questions below.  Instead, you should log on to Blackboard and take the quiz during the time allotted.

Undergraduate Questions

The following questions are for those taking the course for undergraduate credit:
  1. What is pairwise sequence alignment used for?
  2. How do RNA and DNA composition differ?
  3. What model did Watson & Crick suggest?
  4. How is DNA sequenced?
  5. What is "back-translation?"
  6. Use the global alignment algorithm to align:  ACCTGT and CCATGAAT.  What is the score of the optimal alignment?
  7. Repeat above for local alignment.
  8. Which is harder, aligning proteins or aligning DNA?
  9. What's the difference between FASTA and BLAST

Graduate Questions

In addition to the undergraduate questions, those taking the course for graduate question should also do the following questions:

  1. Which is easier, local or global alignment?
  2. What is the Blosum50 matrix?  How was it created?  How does it differ from the Blosum62 matrix?
  3. Is it possible to have more than one optimal local alignment for a pair of sequences?  If yes, give an example.  If not, explain why.
  4. Is it possible to have more than one optimal global alignment for a pair of sequences?  If yes, give an example.  If not, explain why.
